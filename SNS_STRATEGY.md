# HACOS SNS発信戦略

> 「滞っている」を「仕組みで動く」に変える

---

## 1. なぜ今SNSが重要か

- スペースシェアパートナーへの声かけ → SNSが実績・世界観の証明になる
- ドロップイン10人以上 → SNSなしでは届かない
- HACSの「生き物感」= 発信が止まると存在が薄れる

**SNSは集客ツールではなく、HACSという場の「呼吸」**

---

## 2. プラットフォーム戦略

### メイン：Instagram（最優先）

| 機能 | 用途 |
|------|------|
| フィード投稿 | 世界観・トレーニング・仲間の紹介 |
| リール | 動く・伝わる・拡散される |
| ストーリーズ | 日常・イベント告知・裏側 |
| ハイライト | HACSとは／パートナー紹介／イベント記録 |

### サブ：TikTok（リーチ拡大用）
- Instagramリールを転用でOK（手間ゼロ）
- 新規層への入口として活用

### コミュニティ管理：LINE公式 or グループLINE
- 既存会員・ドロップイン参加者への告知
- SNSを見ていない人への確実な届け方

---

## 3. コンテンツの5本柱

HACSらしい発信を「何を投稿すればいいかわからない」にしないための軸。

```
柱1：【動く】Movement
  トレーニング動画・技術Tips・ワークアウト記録

柱2：【整える】Recovery & Care
  ストレッチ・ボディケア・ヨガ・よもぎ蒸しなど
  ← パートナーとの共同コンテンツ起点

柱3：【仲間】Community
  パートナー紹介・会員ストーリー・スタッフ日常
  ← HACSが「生き物」に見える核心

柱4：【体験】Experience
  イベント告知・当日レポート・参加者の声
  ← ドロップイン10人以上を実現する柱

柱5：【場所】Space & Vibe
  朝のHACS・空間・器具・空気感
  ← 来たことない人に「行きたい」と思わせる
```

## 5. 今すぐ再起動するための「3ステップ**

### Step 1：まず1本リールを出す（今週中）
滞ってる間の「復帰宣言」は不要。何事もなかったように動画1本出す。

**おすすめ最初の1本：**
```
「たかや神社 登山ドロップイン」の告知リール
・景色・自然・動く身体
・参加費3,000円・日付
・「詳細はストーリーズで」
```
→ これだけでドロップイン集客も同時進行

---

### Step 2：パートナーとの「タグ付け連携」を始める
- ヨガ・よもぎ蒸し・整体のパートナーをHACSの投稿にタグ付け
- パートナーにもHACSをタグ付けして投稿してもらう
- お互いのフォロワーに自然に届く＝実質無料広告

---

### Step 3：イベントを「コンテンツ」として撮り続ける
- ドロップイン当日は必ずリール素材を撮る（30秒でいい）
- 参加者の笑顔・動いている瞬間・自然の景色
- 投稿するだけで次回の集客になる

## 6. ドロップイン10人以上を実現するSNS戦略

### 現状：3〜5人 → 目標：10人以上

| アクション | 効果 | 難易度 |
|-----------|------|--------|
| 自分のSNSで告知（リール）| 既存フォロワーに届く | ★低 |
| パートナー全員がシェア | 各自のフォロワーにも届く | ★低 |
| 参加者がストーリーズでシェア | 口コミ拡散 | ★低 |
| イベント後レポート投稿 | 次回への期待を高める | ★低 |
| LINE公式で既存会員に告知 | SNS見てない人にも届く | ★低 |

**パートナーが4人いれば、各自500フォロワーでも合計2,000人にリーチできる。**

### 数字の目標設定

```
第1回ドロップイン（登山）：5〜8人
  → 参加者全員がシェアする場をつくる
  → 「楽しかった」投稿が自然に出る演出を

第2回：8〜12人
  → 前回参加者が友達を連れてくる設計

第3回以降：10〜15人定着
  → リピーター＋新規の循環が生まれる
```

## 7. コンテンツ制作を「楽にする」仕組み

### バッチ制作（まとめ撮り）
- 週1回30分、スマホで素材を撮りためる
- 撮影日を決める（例：月曜の開店前）
- 1日の撮影で1週間分の素材が揃う

### パートナーに任せる部分
- よもぎ蒸し・整体・ヨガ → それぞれ自分のサービス動画を撮ってもらう
- HACSのアカウントでリポスト or コラボ投稿
- 自分が全部作らなくていい

### テンプレ化する
- イベント告知の投稿フォーマットを固定
- キャプションの書き方をパターン化
- 「何を書けばいいか」の迷いをなくす

## 8. キャプション・ハッシュタグ戦略

### 基本キャプション構成
```
1行目：目を引くひとこと（「朝7時のHACS」「汗をかく日曜日」）
2〜3行：内容・想い（短く）
CTA：「詳細はストーリーズで」「DMください」「プロフリンクから予約」
ハッシュタグ：5〜10個
```

### ハッシュタグ（テンプレ）
```
メイン：#HACOS #ハコス
ジム系：#パーソナルトレーニング #セミパーソナルジム
地域系：#[地域名]ジム #[地域名]トレーナー
体験系：#ドロップイン #ヨガ体験 #整体 #よもぎ蒸し
ライフスタイル：#朝活 #ボディメイク #体を整える
```
