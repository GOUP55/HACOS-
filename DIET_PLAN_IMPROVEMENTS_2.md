## #4 「15〜20万円払う勇気」を生む購入設計

### 現状の課題
15〜20万円は大きな意思決定。今のプランには：
- 「効果がなかったらどうしよう」というリスクを消す設計がない
- 無料カウンセリング → いきなり本契約の落差が大きすぎる
- 分割払い・段階的な入口がなく、比較検討の時間に流出している

### 改善案

#### A. 「1ヶ月体験版」を入口に設計する
```
体験プラン（¥45,000 / 1ヶ月）
├── PT × 8回（週2回）
├── 栄養カウンセリング × 1回
├── 体組成測定 × 2回（開始・終了）
└── LINEサポート 30日間

→ 1ヶ月後、本コース申込でこの¥45,000を差引きオファー
  「気に入ったら続けてください。合わなければそこで終わりです」

効果：
  高額の意思決定を「小さなYES」から始められる
  1ヶ月で体験する成果がそのまま最強の営業になる
```

#### B. 「14日間全額返金保証」で心理的安全を作る
```
条件：
  ・スタートから14日以内
  ・全PTセッション参加 + 食事ログ提出
  ・「期待と違った」理由なら返金（理由不問）

伝え方：
「HACOSは成果に自信があります。
 14日で変化を感じなければ、全額お返しします。
 でも、感じた人は全員続けています。」

実際の効果：
  返金請求はほぼゼロ（コミット条件が自己選択を促す）
  「保証がある」という事実が購入の後押しになる
```

#### C. 分割払いプランで月額換算を前面に出す
```
現状の見せ方：
  「3ヶ月 180,000円」→ 高く見える

改善後の見せ方：
  「月60,000円 × 3ヶ月」
  「1日たったの2,000円。コンビニランチより安い投資です」

さらに分割対応：
  ・3回払い（手数料なし）
  ・6回払い（+5%手数料）
  → ペイディ / GMO後払い等と連携
```

## #5 ダイエット失敗の本当の原因「メンタル」への介入

### 現状の課題
ダイエット失敗の原因の研究では：
- **身体的な問題（プランが悪い）：約20%**
- **精神的な問題（続けられない・感情食い・自己否定）：約80%**

今のプランはフィジカル（運動・食事）を完璧に設計しているが、  
**心が折れたときの設計が一切ない。**

| 見落としている場面 | 何が起きるか |
|-----------------|------------|
| 停滞期（3〜4週目） | 「体重が減らない＝自分のせい」と思い込む |
| 外食・飲み会 | 「もう今日はいいや」からの連鎖崩壊 |
| ストレス過多日 | 感情食いが起きてもサポートなし |
| 成果が出ない自己嫌悪 | 次のセッションに来なくなる |

### 改善案

#### A. 「心理的安全ガイド」をプランに組み込む
スタート時に渡す小冊子（A5・8ページ）。

```
【HACOS メンタルブック】目次

1. 停滞期は「成功のサイン」（生理学的な理由）
2. 「今日やらかした」日のリカバリー3ステップ
3. 感情食いのトリガーを知る（チェックリスト）
4. HACOSトレーナーへの「SOS の送り方」
5. 90日間の「心の波形」予測図
```

「停滞期はこの時期に来ます」と予告するだけで、  
実際に来たときのパニックが大幅に減る。

#### B. 週次PTセッションに「5分間メンタルチェック」を追加
```
セッション最初の5分：トレーナーが毎回聞く3問

Q1「今週、一番しんどかった瞬間はいつですか？」
Q2「逆に、よく頑張ったと思う場面は？」
Q3「今日は何点の状態で来ましたか？（10点満点）」

効果：
  ・クライアントが「受け止めてもらえている」と感じる
  ・7点以下の週は強度を落とす判断ができる
  ・記録を積み上げると成長の軌跡になる
```

#### C. 「感情食いSOSプロトコル」をLINEに設ける
```
深夜に暴食したくなったとき、LINEで送れる仕組み。

クライアント：「SOS」と送信
    ↓
自動返信（即時）：
「気づいて送ってくれてありがとう。
 まず水を1杯飲んでください。
 5分後にまだ食べたかったら食べていいです。
 でも、5分で落ち着いた人が9割います。
 明日、話しましょう。」

※ 翌朝のセッションで必ずフォロー
```

## #6 成果が出たクライアントを「集客資産」に変える仕組み

### 現状の課題
- 頑張って結果を出したクライアントが「個人の成功体験」で終わっている
- 紹介が来るかどうか「その人の性格次第」になっている
- SNS投稿・口コミ・Googleレビューへの導線が一切ない
- 成果事例を積み上げてもアーカイブ・活用できていない

### 改善案

#### A. 「紹介プログラム」を明文化・自動化する
```
紹介した人（既存クライアント）への特典：
  ・次回PTセッション 1回無料（¥6,000相当）
  ・アフターケアメンバーシップ 1ヶ月 ¥2,000引き

紹介された人（新規クライアント）への特典：
  ・初回カウンセリング + 体組成測定 無料（通常¥3,000）
  ・体験1ヶ月プラン ¥3,000引き

仕組み：
  卒業セッション時に「紹介カード」を3枚手渡す
  → 渡すだけでOK（相手が使ったら自動的に特典発生）
```

#### B. 「Before/After コンテンツ」製造ラインを作る
90日かけて結果が出た瞬間こそ最大のコンテンツ。これを逃さない。

```
Week 1（開始時）
  ├── 体組成測定データ記録
  ├── 「今の自分をどう思いますか？」インタビュー（LINE文章）
  └── Before 写真（希望者のみ・完全非公開でもOK）

Week 12（卒業時）
  ├── 体組成測定データ記録（比較レポート）
  ├── 「3ヶ月でどんな変化がありましたか？」インタビュー
  └── After 写真（希望者のみ）

コンテンツ展開：
  ・Instagramストーリーズ（顔なし・数字だけでも強力）
  ・ハイライトに「お客様の声」として保存
  ・次の体験相談者へのLINE共有
```

#### C. Googleマップ口コミを「仕組みとして」依頼する
```
タイミング：卒業セッションの最後

言い方のテンプレート（口頭）：
「〇〇さんの3ヶ月、本当に素晴らしかったです。
 一つお願いがあって、Google に感想を書いてもらえますか？
 30秒で書けるので、もしよければ今ここで（スマホで）。」

QRコードカードを手渡す（GoogleマップのURLをQRに変換するだけ）

効果：
  今はゼロ件のGoogleレビューが積み上がる
  → 検索で「観音寺市 パーソナルトレーニング」で比較された時に圧勝
```
