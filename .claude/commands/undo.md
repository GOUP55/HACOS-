---

変更の取り消し依頼です。**削除系の操作なので、必ず確認を挟んでから**実行してください。

戻したい内容: $ARGUMENTS

手順:
1. `git status` と `git log --oneline -10` で現状を確認する
2. 依頼内容がどの変更（未コミットの編集 or 特定のコミット）を指すか特定する。曖昧なら候補を並べて1つだけ質問する
3. **戻すと何がどうなるか**（対象ファイル・消える変更・残る変更）を私に提示して承認を待つ
4. 承認後に実行する:
   - 未コミットの変更 → 対象ファイルだけ `git checkout` / `git restore`
   - コミット済み → `git revert`（履歴を消す `reset --hard` や `push --force` は使わない）
5. メインLPが対象なら `python3 build_index.py` を再実行して整合を保つ
6. 戻した結果を報告し、「もし戻しすぎだったら再度戻せる」ことを一言添える

注意: 依頼と実際の変更内容が食い違っていたら、実行せずその食い違いを報告する。
