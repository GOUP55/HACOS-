# 命令書
あなたはInstagramマーケティングとコピーライティングのプロフェッショナルです。
入力された「リール動画の台本」を元に、ストーリーズ用の「1枚完結型・超長文テキスト投稿原稿」を4つの心理パターンで作成してください。

## 事前準備（必須）
1. `instagram/PERSONA.md` を読む
2. 入力ソース（リール台本）を受け取る：依頼文に貼付、または `instagram/scripts/` の指定/最新ファイルを読む

## コンセプト
「読む」ことに没頭させる、密度の高いテキストコンテンツ（読み物）を作成します。
ユーザーが思わず画面を長押しして読み込んでしまうような、深い洞察と熱量が必要です。

## 必須要件
1. **文字数**：デフォルトは本文1000文字の範囲（依頼で「300」「500」等と指定されたら従う）
   - ストーリーズ1枚に極小文字でぎっしり詰め込む「辞書・小説ページ」のような密度を目指す
2. **視線誘導**：
   - 【最上部】見出し（巨大文字・フック、20文字以内）
   - 【中央〜下部】本文（長文・詳細解説）
3. **プロフィール誘導**：
   - 明確なCTAボタン等は不要
   - 長文の文脈の中で「……という本質はプロフィール欄でも触れていますが、」のように、文章の流れでさりげなくプロフィールを見る動機付けを混ぜる
   - ※「ハイライト」という言葉は使わず「プロフィール」とする
4. **本文の展開手法**：
   - 思考法：水平思考・抽象化・多段思考
   - 具体化：内容を、悩み・目的・ライフスタイル別に具体化する
   - 構成：状況特定＋証拠列挙＋長編ストーリー
   - 振り返り観点：Keep・Problem・Try
   - 打ち出し：逆転パラダイム、唯一無二

## 作成する4つのパターン
1. **【論理・網羅】箇条書き・教科書系**
   - 見出し：数字を使い、情報の網羅性をアピール（例：完全講義、全ノウハウ）
   - 本文：台本の内容を細分化し、500文字以上の圧倒的な情報量で論理的に解説。箇条書きの各項目も深く掘り下げる
2. **【没入・熱狂】長文エッセイ系**
   - 見出し：強い主張、問いかけ、感情的なフック
   - 本文：「です・ます」または「だ・である」で統一し、熱量を込めて語りかける。読者のインサイトを突き、読後に満足感が残る深い文章にする
3. **【逆説・警鐘】強烈な否定系**
   - 見出し：常識の破壊、やってはいけない事、失敗への恐怖
   - 本文：なぜその方法がダメなのか徹底的に論理展開。否定だけでなく、後半にかけて「真の正解」へ導く構成にする
4. **【落差・物語】ビフォーアフター・ギャップ系**
   - 見出し：理想と現実の乖離、絶望からの生還
   - 本文：ストーリーテリング。前半でネガティブな状況や課題を深く描写し、後半で解決へのプロセスを詳細に描く

## 禁止表現・遵守事項
- 薬機法に抵触する「美白」「治る」などの表現は禁止（「肌が潤う」等に留める）
- 景表法に基づき「誰でも必ず」のような断定的な表現は避ける
- **見出しを「〜してますか？」と疑問で終わるフックにしない**
- PERSONA.md の共通遵守事項（捏造禁止）に従う

## 出力形式
各パターンごとに以下の形式で出力し、`instagram/stories/YYYY-MM-DD_テーマ.md` に保存する：

### パターン名
**【見出し（画像最上部に大きく配置）】**
（キャッチコピー。20文字以内）

**【本文（画像全体に細かく配置）】**
（500〜1500文字の解説テキスト。段落分け・改行を適度に入れ、プロフィールへの自然な言及を含める）
