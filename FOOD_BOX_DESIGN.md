## ◆ レシピカード設計

**フォーマット：** A5サイズ・両面印刷
**デザインテーマ：** 和・ナチュラル・讃岐らしさ
**内容構成：**

```
【表面】
・レシピ名（大きく）
・難易度・所要時間
・材料（4人分）
・作り方（写真付き3〜5ステップ）

【裏面】
・このレシピのHACOS的ポイント
・食材の豆知識（発酵食の効果など）
・GOからのひと言
・生産者の写真と一言（あれば）
```

## ◆ 梱包・パッケージング設計

### BOXデザイン

**外箱：** クラフト紙（ナチュラル素材）・HACOSロゴ入り
**サイズ：** A4対応（基本BOX）・B4対応（プレミアムBOX）
**テープ：** HACOSロゴ入りクラフトテープ
**緩衝材：** 再生紙・新聞紙（プラスチック不使用）

### 開封体験の設計

```
開封の順序：
1. 上部にHACOS通信（ニュースレター）
2. レシピカードが封筒に入って置いてある
3. 食材は種類ごとに小分け袋に入って整列
4. 底にGOの手書きシール or スタンプ
5. 冷蔵品は保冷剤付き
```

## ◆ 展開ロードマップ

### Phase 1（今すぐ）：コホート参加者向け付帯サービス

- スタンダード・本気プランの「食サポートFull」に月1BOXを含める
- 追加コストなし（プラン料金内）
- 5名分から始める → 品質と物流を確認

### Phase 2（3ヶ月後）：単品販売開始

- Instagramから「HACOS食材BOX購入者」を募集
- LINE公式で注文受付
- 月30箱を目標

### Phase 3（6ヶ月後）：サブスクモデル本格展開

- ECサイト（BASE / STORES）でサブスク販売
- 定期便割引導入
- 「食材BOX＋オンラインサポート」の新プラン展開

### Phase 4（12ヶ月後）：全国展開

- 送料改善（ヤマトとの契約）
- 産地直送オプション追加
- メディア露出連動で一気に認知拡大
