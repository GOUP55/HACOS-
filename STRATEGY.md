# HACOS ビジネス戦略

> **「HACOSを唯一無二へ成長」**
> 思い・信念・ブランドは変えない。構造だけをゼロから再構築する。

---

## 1. HACOS とは何者か（不変の核）

### ミッション
**「箱（ハコ）を持たない人たちが、ここで輝ける場所をつくる」**

### ビジョン
- フリーランストレーナー・サービス提供者が集まる「生きたコミュニティ」
- 利用者もトレーナーも、HACSに関わることで成長できる場
- カフェ・ジム・コミュニティが有機的につながる複合空間（長期）

### バリュー
- **唯一無二** ── 他にない体験・空間・コミュニティ
- **共創** ── トレーナー・利用者・事業者が一緒につくる
- **生き物** ── 固定されず、常に進化し続ける

---

## 2. 現状分析（2026年4月時点）

### 強み
- 火〜金：パーソナルトレーニング **約70%稼働**（安定した収益基盤）
- 物理的な「箱（スペース）」を保有
- ブランドと想いの軸がある

### 課題
| 課題 | 詳細 |
|------|------|
| 人材 | メイントレーナー1名のみ（属人化リスク）|
| 稼働日 | 日曜・月曜が未活用 |
| カフェ事業 | 撤退または週2回営業（不確定）|
| コミュニティ | 仲間・メンバーがまだいない |

### 機会
- フリーランストレーナーの増加（箱を持てない人が多い）
- セミパーソナル需要の拡大
- 日曜ドロップインでの新規顧客獲得

## 3. ゼロベース再構築の方針

**捨てること：** カフェへの依存、一人で全部やる構造  
**残すこと：** ブランド・思い・現在の稼働基盤  
**作ること：** コミュニティ・シェア収益モデル・日曜コンテンツ

### 事業の3本柱（再構築後）

```
HACOS
├── 柱1：パーソナル/セミパーソナルジム（既存・強化）
│   火〜金 稼働中 → 土曜も検討
│
├── 柱2：コミュニティスペース（新設）
│   フリーランストレーナー・サービス提供者のシェアスペース
│   収益：スペースレンタル料 or 売上シェア
│
└── 柱3：日曜ドロップインプログラム（新設）
    一般参加型コンテンツ
    コミュニティ形成の入口
```

## 4. 短期戦略（〜3ヶ月：2026年4〜6月）

### 優先課題：仲間づくり

**Step 1：仲間の定義を決める**
- どんなトレーナー・サービス提供者を求めるか
- HACSのバリューに合う人材像の言語化

**Step 2：日曜コンテンツの設計**
- 何を提供するか（ドロップインクラス、体験会、コミュニティイベント）
- 誰が担当するか（自分 or 仲間トレーナー）
- 集客方法（SNS・紹介・既存会員への告知）

**Step 3：シェア条件の整備**
- スペース利用ルール
- 料金体系（時間貸し / 売上シェア / 月額）
- 最初の1〜2名を試験運用

### 今月のアクション（4月）
- [ ] 仲間トレーナーの条件・プロフィールを言語化
- [ ] 日曜ドロップインの第1回コンセプトを決める
- [ ] スペースシェアの仮条件を作る

## 5. 中期戦略（3〜12ヶ月：2026年7月〜2027年3月）

### コミュニティの自走化
- 3〜5名のフリーランサーがHACSを「ホームジム」として活用
- 日曜コンテンツが月1→月4回へ拡大
- 会員（ファン）がトレーナーを紹介してくれる循環が生まれる

### カフェ連携（不確定要素への対応）
- カフェが戻る場合：コミュニティの「たまり場」として統合
- カフェが来ない場合：軽食・ドリンクサービスを別途検討（自販機・デリバリー提携）

### 収益目標イメージ
| 収益源 | 現在 | 6ヶ月後目標 |
|--------|------|------------|
| パーソナルトレーニング | ●●万円 | 維持〜+10% |
| スペースシェア | 0 | +●万円 |
| 日曜ドロップイン | 0 | +●万円 |
| 合計 | ●●万円 | +●●万円 |

*※ 数字はファイナンス担当と連携して記入予定*

## 6. コミュニティ設計

### 関わる人のレイヤー

```
レイヤー1：コアメンバー（トレーナー・サービス提供者）
  → 定期的にHACSを使うフリーランサー
  → HACSのブランドを一緒に育てる仲間

レイヤー2：レギュラー会員
  → 週1〜複数回通う固定メンバー

レイヤー3：ドロップイン参加者
  → 日曜などの単発利用者
  → コミュニティへの入口

レイヤー4：SNSフォロワー・ファン
  → オフライン参加はまだだが、応援してくれる人
```

### コミュニティに集めたいサービス提供者（案）
- パーソナルトレーナー
- ヨガ・ピラティスインストラクター
- 栄養士・食事コーチ
- ボディケア・マッサージ
- メンタルコーチ・コーチング
- その他：コンセプトに合うもの
