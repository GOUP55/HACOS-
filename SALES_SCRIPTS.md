## 2. 無料カウンセリング（45分）進行スクリプト

### オープニング（5分）
```
「今日は時間を作ってくれてありがとうございます。
 今日の流れなんですけど、
 まず〇〇さんのことをもっと知りたくて、
 20分くらい話を聞かせてください。
 その後、体組成を測って、
 最後に私からHACOSのプランをお伝えします。
 強引に勧めたりしないので、
 気になることは何でも聞いてください。」
```

### ヒアリング（20分）
```
【ライフスタイルスコア 5問】
Q1「平日、自分のための時間って1日どれくらいありますか？」
Q2「週に外食や飲み会って何回くらいありますか？」
Q3「パートナーや家族はあなたのダイエット、応援してくれそうですか？」
Q4「これまで挫折したとしたら、一番の理由って何でしたか？」
Q5「ひとりで黙々と頑張るのと、仲間と一緒に頑張るの、どっちが合ってますか？」

【睡眠スクリーニング 3問】
Q6「平均、何時に寝て何時に起きてますか？」
Q7「朝起きたとき、スッキリしてると感じますか？（5段階で）」
Q8「夜中に目が覚めることはありますか？」

【5つのなぜ（本当の動機を掘り下げる）】
Q9「ダイエットしたい、一番の理由ってなんですか？」
→「なぜそれが大事ですか？」を最大5回繰り返す
→ 最後に「それが実現したとき、どんな自分でいますか？」
→ 出てきた言葉をメモ（プランブックに書く）
```

### 体組成測定（5分）
```
「じゃあ体組成を測りましょう。
 数字を出すのが目的じゃなくて、
 今の状態を一緒に把握するためです。
 結果に良い悪いはないので、気楽にいきましょう。」

（測定後）
「見てほしいのはここです。
 体重より、体脂肪率と筋肉量のバランス。
 〇〇さんの場合、ここが変わると
 見た目がガラッと変わります。」
```

### プレゼン（10分）
```
「聞かせてもらったことをもとに、
 〇〇さんには [LIFE / EXEC / FREE] タイプが合いそうです。

 HACOSのプランって、
 運動・食・整えるの3つを全部一緒にやります。
 でも一番大事にしてるのは、
 90日間、途中で諦めない設計をすること。

 [ライフスタイルに合わせた特徴を1〜2点説明]

 料金は月 [価格] です。
 高いと感じるかもしれないんですけど、
 これ、他のジムで同じ内容受けようとすると
 [相場比較を一言で] かかります。

 14日間、合わなければ全額返金もしています。
 だから、まず試してみてほしいんです。」
```

### クロージング（5分）
```
「今日どう感じましたか？
 正直に教えてもらえると嬉しいです。」

（「やってみたい」なら）
「じゃあ、スタート日を決めましょう。

 決めてもらえると助かります。」

（「お金が…」なら）
「分割払いもできますよ。
 3回払いなら手数料なしで、
```

## 3. LINEフォローアップスクリプト

### カウンセリング翌日（まだ返事がない人へ）
```
「昨日はありがとうございました！
 話してみてどうでしたか？
 何か気になってることがあれば
 遠慮なく聞いてください😊」
```

### 3日後（まだ保留の人へ）
```
「〇〇さん、こんにちは。
 先日の話、どんな感じですか？

 ひとつお伝えしたいことがあって、
 来月のコホートの枠が
 残り [○名] になってきました。
 
 決めやすいように何かできることがあれば
 なんでも言ってください☺️」
```

### 1週間後（最終フォロー）
```
「〇〇さん、最後に一度だけ。

 もしHACOSが合わないと思うなら
 それは全然いいんです。
 ただ、もし迷ってるなら、
 一歩踏み出す勇気が一番難しいのに、
 そこまで来てくれてるんだから、
 もったいないなと思って。

 何か手伝えることがあれば
 いつでも声かけてください。」
```
