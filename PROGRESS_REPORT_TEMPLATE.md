## 2. 週次LINEチェックイン（毎週月曜 自動送信文）

> Google フォームのURLと一緒に送る。

```
【送信文】

おはようございます☀️
今週もHACOS週が始まります！

3問だけ答えてください（1分でOK）
↓
[Googleフォームのリンク]

今週のミールプランも送ります。
いってきます！ GO
```

**Googleフォームの設問：**
```
Q1「先週の睡眠の質は？」（1〜5のスケール）
Q2「食事の自己評価は？」（1〜5のスケール）
Q3「先週のベストな選択は何でしたか？」（自由記述）
Q4「今週乗り越えたいことはありますか？」（自由記述・任意）
```

## 3. コホートLINEグループ 週次投稿テンプレート

### 月曜（目標宣言）
```
GO：
「今週もスタート🔥
 それぞれ今週の目標を1行で宣言してください！
 例）「今週は朝ごはんを絶対食べる」
 例）「水を1日2L飲む」
 じゃあ俺から。今週の目標：[GOの目標]」
```

### 金曜（ベスト報告）
```
GO：
「今週お疲れ！
 今週のベストな選択を1つシェアしてください🎉
 小さいことでOKです。
 例）「飲み会でサラダ先食べした」
 例）「雨でも走った」」
```

### 測定週（隔週水曜）
```
GO：
「今週末は測定です！
 数字より、この2週間でどんな変化を感じたか
 教えてください😊
 体だけじゃなく、気持ちの変化も大歓迎。」
```

## 5. 停滞期予防接種 封筒テンプレート

> Week 1のセッション後に手渡す。封筒の外側に書く文言と、中の手紙。

**封筒外側：**
```
4週目（約30日後）に開けてください。
それより前は開けないでください。
      ── GO
```

**封筒内側の手紙：**
```
〇〇さんへ

今、体重が思うように落ちていないかもしれません。

でも聞いてください。

体脂肪が減る前に、体の中は変わっています。
ホルモンが整い、代謝が上がり、筋肉が育っている。
それが体重という数字に出るのが、少し遅れているだけです。

それに。

あなたが最初に教えてくれた言葉を覚えていますか？

「___________________________________」

体重の数字は、あなたの全てじゃない。
あなたが来た理由は、もっと大きい。

明日もここに来てください。
それだけでいいです。

                    ── GO

追記：
もしこれを読んで「辛い」と感じているなら、
今すぐLINEで「SOS」と送ってください。
```
