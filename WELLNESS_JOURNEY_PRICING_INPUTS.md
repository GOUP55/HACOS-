## 5. 計算構造（数字が埋まったらこの式で回す）

```
売上(n名)   = S8 × n ＋ オプション売上
変動費(n名) = (C1×5 + C2×5 + C3×n食 + C4 + C5 + C6 + C8) × n ＋ S8×n×C7
準固定費    = F1 + F2×グルトレ回数 + F3 + F4 + F5 + F6
利益(n名)   = 売上 − 変動費 − 準固定費
チェック: 利益(3名) ≥ 0 かつ 利益(5名) ≥ P2
```

## 6. 仮定シミュレーション（⚠️ 全て仮定の数字。意思決定には使わず、構造の確認用）

> 参加費 ¥45,000、食事変動費 ¥9,000/人、決済 3.6%、GO報酬 ¥25,000/期、
> トレーナー謝礼 ¥5,000×2回、調理人件費 ¥10,000/期、運営費 ¥5,000/期 と**仮定**した場合:

| | 3名 | 4名 | 5名 |
|---|---|---|---|
| 売上 | ¥135,000 | ¥180,000 | ¥225,000 |
| 変動費（食事＋決済） | ¥31,860 | ¥42,480 | ¥53,100 |
| 準固定費 | ¥50,000 | ¥50,000 | ¥50,000 |
| **利益** | **¥53,140** | **¥87,520** | **¥121,900** |

構造上の示唆（仮定でも言えること）:
- 準固定費（GO報酬・トレーナー・調理）が¥50,000規模なら、**3名でも黒字にできる構造**
- 人数が1名増えるごとの限界利益は大きい（食材以外ほぼ増えない）→ **4人目・5人目の集客が利益の源泉**
- 参加費を下げるなら、先に食事の点数（夕食セットの日数など）を調整する方が構造を壊さない
