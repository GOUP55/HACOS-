## 0. 収益モデル（07-14決定を反映）

参加費はひとつのパッケージ価格だが、**内部では2つに分かれる**:

```
参加費 ＝ ①運動・サポート部分（事業側の売上）
        ＋ ②食事フルセット部分（みどり/ふみやさんの売上。原価を引いた利益が2人の報酬）
```

- ②「フルセット」＝スムージー×5＋弁当×5＋間食＋冷凍おかずセット×7（範囲は要確認）
- 事業側は食事の原価も人件費も持たない代わりに、②の金額は全額みどり/ふみやさんへ
- **⚠️ 検算ポイント（リスク#14）**: ①だけでGO報酬・トレーナー謝礼・運営費・決済手数料を賄って利益が残るかを必ず確認する
- **最重要の未決定数字**: ②フルセットの内部価格。これが決まらないと参加費が決められない

## 5. 計算構造（07-14の収益モデルで更新）

```
事業側売上(n名)  = (S8 − S11) × n ＋ オプション売上   ※S11はみどり/ふみやさんへ
事業側変動費(n名)= (C5 + C6 + C8) × n ＋ S8 × n × C7   ※決済手数料は参加費全額にかかる
準固定費         = F1 + F2×グルトレ回数 + F4 + F5 + F6
事業側利益(n名)  = 事業側売上 − 事業側変動費 − 準固定費

みどり/ふみや側: 利益 = S11 × n − (C1×5 + C2×5 + C3×7 + C4) × n
チェック: 事業側利益(3名) ≥ 0 ／ 事業側利益(5名) ≥ P2 ／ みどり/ふみや側利益が2人の働きに見合うか
```

## 6. 仮定シミュレーション（⚠️ 全て仮定の数字。意思決定には使わず、構造の確認用）

> 参加費 ¥45,000、うち食事フルセット内部価格 ¥15,000/人、食材原価 ¥9,000/人、決済 3.6%、
> GO報酬 ¥25,000/期、トレーナー謝礼 ¥5,000×2回、運営費 ¥5,000/期 と**仮定**した場合:

| | 3名 | 4名 | 5名 |
|---|---|---|---|
| 事業側売上（食事分を除く） | ¥90,000 | ¥120,000 | ¥150,000 |
| 決済手数料（参加費全額×3.6%） | ¥4,860 | ¥6,480 | ¥8,100 |
| 準固定費（GO＋トレーナー＋運営） | ¥40,000 | ¥40,000 | ¥40,000 |
| **事業側利益** | **¥45,140** | **¥73,520** | **¥101,900** |
| （参考）みどり/ふみや側利益 | ¥18,000 | ¥24,000 | ¥30,000 |

構造上の示唆（仮定でも言えること）:
- この構造なら**3名開催でも両側黒字**にできる。鍵は「S11をいくらにするか」の1点
- 人数が1名増えるごとの事業側限界利益は大きい → **4人目・5人目の集客が利益の源泉**
- みどり/ふみや側の利益は人数に比例する → 2人にも「5名集める」動機が生まれる良い設計
