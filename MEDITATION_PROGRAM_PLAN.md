## おかちんと決めること（LINE打ち合わせアジェンダ）

優先度順。上の3つが決まらないと告知物が作れない。

1. **開催日時と頻度** … 単発のお試し開催から始めるか、月1などの定期か。曜日・時間帯（朝活と被らないか）
2. **会場の確定** … 宗林寺の本堂・客殿など具体的にどこで行うか。駐車場の有無・台数
3. **定員と最少催行人数** … 写経道具の数・スペースで上限が決まるはず
4. **¥4,000の内訳と配分** … お寺への謝礼・会場費・道具代（筆・墨・写経用紙）をどう分けるか。道具はどちらが用意するか
5. **対象** … 一般公開か、HMC会員・LINE友だち優先か。初心者前提でよいか。服装（正座が難しい人向けに椅子があるか）
6. **予約と支払い** … 予約は公式LINEで受けるか（LINE予約システムに組み込むかは規模を見てから）。支払いは現金かSTORESか（既存ルール上Stripeは使わない）
7. **キャンセルポリシー** … 前日・当日キャンセルの扱い
8. **写真・SNS掲載** … 境内・本堂の撮影可否、参加者が写る写真の扱い（告知素材に必要）
9. **宗教面の伝え方** … 宗派の行事ではなく「どなたでも参加できるマインドフルネス体験」であることをどう表現するか（おかちんの意向を確認）
