## プロジェクト概要

あなたは HACOS LIFESTYLES（オーナー：GO）の事業設計・実装サポートをしている。

**HACOS とは：**
- 観音寺市・香川県にあるパーソナルトレーニングジム
- 運動 × 食（讃岐発酵食哲学）× 睡眠 を統合した90日ダイエットプランを展開
- コホートモデル：同じ日にスタートする3〜5名が一緒に90日を歩む
- オーナーは「GO」（35歳・観音寺在住）

**重要：場所は観音寺市（丸亀市ではない）**

## ビジネス哲学・ブランドの核心

```
食の3原則：
  01「引く食より足す食」（カロリー制限より良いものを足す）
  02「土に近いものを食べる」（地元・旬・加工度低）
  03「食べる行為を整える」（よく噛む・感謝・一人で食べない）

コホートの価値：
  「ひとりで続けられない理由は、意志が弱いからじゃない。ひとりだから。」

GOのブランド：
  「讃岐で育てられた食哲学を持つトレーナー」
  35歳・観音寺在住・発酵食・地元農家との連携
```

## 90日プログラム構造

```
Phase 1（Week 1〜4）：習慣の種まき期
  毎週：月曜 目標宣言 → 金曜 ベスト報告
  隔週：体組成測定（Week 2・4）
  Month 1：発酵食ワークショップ「塩糀仕込み」
  Week 1：ウェルカムキット手渡し・5つのなぜ・4週封筒
  Week 4：4週封筒開封

Phase 2（Week 5〜8）：強化と定着期
  Body：有酸素+筋力トレーニング強化
  Food：外食・飲み会のマネジメント
  Month 2：発酵食ワークショップ「味噌汁再設計」
  体組成測定（Week 6・8）

Phase 3（Week 9〜12）：仕上げと自立準備期
  卒業後の生活設計
  Month 3：発酵食ワークショップ「酵素シロップ」
  体組成測定（Week 10・12）
  Week 12：Afterインタビュー・Googleレビュー・次期案内
```
