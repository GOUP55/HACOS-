## コンビニ 最強セット例

### セブン-イレブン
```
◎ タンパク質を摂りたいとき
  サラダチキン（プレーン）+ ゆでたまご + 野菜スープ
  → タンパク質 約35g、コスト約¥500

◎ 朝ごはんに
  ギリシャヨーグルト + ゆでたまご + 温野菜サラダ
  → タンパク質 約25g、腸活もOK

◎ ランチに
  豆腐とサラダチキンのサラダ + 具だくさん味噌汁
  → 血糖値が上がりにくい組み合わせ
```

### ファミリーマート
```
◎ タンパク質を摂りたいとき
  サラダチキンバー + ゆでたまご2個 + 野菜スムージー
  → タンパク質 約32g

◎ 甘いものが食べたいとき
  ギリシャヨーグルト（無糖）+ ナッツ
  → 砂糖の代わりに良質な脂質と食物繊維

◎ 夜遅い日に
  豆腐バー + 具だくさん汁物
  → 消化に負担をかけずタンパク質確保
```

### ローソン
```
◎ ブランパン系が充実（糖質を抑えたいとき）
  ブランパン + ゆでたまご + サラダ
  
◎ 鶏むね肉系惣菜 + 野菜スープ
  → シンプルで使いやすい
```

## 絶対に避けたいもの

```
× 菓子パン・惣菜パン
  → 糖質＋脂質のダブルパンチ。腹持ちも悪い

× 揚げ物（フライドチキン・コロッケ等）単品
  → 油の質が悪く、食べすぎを招く

× 甘い缶コーヒー・清涼飲料水
  → 1本で角砂糖10個分の糖質

× 菓子・スナック（食事の代わりに）
  → タンパク質ゼロで血糖値だけ上がる
```

## 外食 選び方のルール

### ランチの鉄則
```
1. 最初に野菜 or 汁物から食べる
   → 血糖値スパイクを防ぐ（これだけで体脂肪のつき方が変わる）

2. 「ご飯少なめ」を必ずオーダーする
   → ほとんどの店で対応してくれる。無料

3. 揚げ物より「焼く・蒸す・煮る」を選ぶ
   → 焼き魚定食 > 唐揚げ定食
```

### 定食屋・和食
```
◎ 焼き魚定食（ご飯少なめ）
◎ 刺身定食
◎ 豚汁定食
◎ 蒸し鶏や湯豆腐のある店

△ 天ぷら・フライ（週1回まで）
× カツ丼・天丼（特別な日だけ）
```

### 中華・ラーメン
```
◎ チャーハン ＜ 炒め物系（油少なめ頼む）
◎ ラーメン食べるなら：
   スープ半分残す + 麺少なめ + 追いたまご
△ 週2回以内に抑える
```

### イタリアン
```
◎ サラダ先食べ → メイン（魚 or 肉）→ パスタ少量
◎ ピザより「魚のグリル」を優先
× パスタのみ、ピザのみは糖質過多
```

### 焼肉・居酒屋
```
◎ 焼肉：タン・カルビより → ロース・ハラミ・鶏肉が◎
◎ 居酒屋のおすすめ：
   枝豆・だし巻き・刺身盛り・焼き鳥（塩）・冷奴
◎ お酒を飲むなら：ハイボール or 焼酎水割り（ビール・甘いサワーは控えめに）
× 締めの炭水化物は飛ばせるなら飛ばす
```

## 飲み会・接待サバイバル術

```
前日：タンパク質を多めに摂っておく（筋肉を守る）

当日：
  ① 飲み会前に何か食べてから行く（空腹で乗り込まない）
  ② 最初の1杯目は好きなものを飲んでいい
  ③ 2杯目以降 → ハイボール or 水or烏龍茶を挟む
  ④ つまみは枝豆・刺身・焼き鳥（塩）を中心に
  ⑤ 揚げ物は食べてもいいが1〜2個まで

翌朝：
  白湯500ml → 普通の食事 → 普通に動く
  「昨日の分を取り返そう」は絶対NG
```

## タンパク質クイックリファレンス

```
目標：体重 × 1.5g/日（例：60kgなら90g/日）

食材とタンパク質量（目安）：
  鶏むね肉 100g → 約23g
  鶏もも肉 100g → 約17g
  卵 1個   → 約6g
  豆腐 1丁  → 約20g
  サラダチキン1袋 → 約25g
  プロテイン1杯  → 約20〜25g
  刺身盛り1人前  → 約20〜30g

1日のイメージ：
  朝：卵2個（12g）+ ヨーグルト（10g）= 22g
  昼：チキンサラダ（25g）+ 豆腐（10g）= 35g
  夜：魚 or 肉（25g）+ 味噌汁（5g）= 30g
  合計：87g → ほぼ目標達成
```
