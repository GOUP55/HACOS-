# HMC コピーライティング & メッセージングガイド
## Healthy Morning Club — Words That Move People

---

## 1. ブランドの声（Voice & Tone）

### HMCが話す「声の人格」
HMCのコピーは、**信頼できる友人が、静かに、でも確信を持って話しかける**ような声。
- 説得しない。体験へ誘う。
- 効能を語らない。感覚を描写する。
- 命令しない。「こっちにおいで」と手を差し伸べる。

### トーン別ガイドライン
| 場面 | トーン | NG例 | OK例 |
|------|--------|------|------|
| SNS告知 | 詩的・体感的 | 「参加者募集！」 | 「月曜の朝を、変えにきてください」 |
| LP本文 | 問いかけ・共感 | 「健康になれます」 | 「今週、自分の感覚を信じましたか」 |
| 予約完了メール | 温かく・儀式的 | 「ご予約ありがとうございます」 | 「あとは、スマホを置いてきてください」 |
| 当日アナウンス | 静かに・確信 | 「ルールを守ってください」 | 「今日の90分は、あなたのものです」 |
| フォローアップ | 余韻を大切に | 「次回もよろしく」 | 「今日の感覚を、少し持ち帰ってください」 |

## 3. 体験柱別コピーバリエーション

### 柱①「ご褒美ヘルシー」
```
【案A】 「美しいものが、体にいい。」
【案B】 「今日の朝食は、シェフが選んでくれる。」
【案C】 「我慢じゃなくて、恵みとして食べる朝。」
【案D】 「食事が、アートになる朝がある。」
```

**使い分け:**
- Instagram投稿: 案A（シンプルで映える）
- メニュー紹介ページ: 案C（感情的な転換）
- フード写真のキャプション: 案B（人の存在を感じさせる）

### 柱②「感覚のコントラスト」
```
【案A】 「極限の熱が、頭の中を空白にする。」
【案B】 「サウナの後のアイスバスで、別人になる朝。」
【案C】 「身体のスイッチを、強制的に切り替える。」
【案D】 「熱くなって、冷えて、整う。それだけでいい。」
```

**使い分け:**
- サウナ体験の告知: 案B（体験を具体的に描写）
- 体験全体の説明: 案C（価値訴求）
- 体験後レビュー的な文脈: 案D（シンプルな事実）

### 柱③「ムード・チューニング」
```
【案A】 「音が、自律神経を整えていく。」
【案B】 「シンギングボウルの振動で、脳がリセットされる感覚。」
【案C】 「お茶一杯に、今ここに戻ってくる方法がある。」
【案D】 「静と動の両方があるから、本当に整う。」
```

**使い分け:**
- シンギングボウル体験の説明: 案A・B
- マインドフルネス（おさちさん）の告知: 案C
- 体験全体の説明: 案D

### 柱④「選択からの解放」
```
【案A】 「スマホを置いてくることが、最高の贅沢になる朝。」
【案B】 「今日だけは、アルゴリズムに従わなくていい。」
【案C】 「何も決めなくていい90分。すべてHMCが設計する。」
【案D】 「AIがいない場所で、自分の感覚だけを信じる。」
```

**使い分け:**
- コンセプト説明ページ: 案B（時代性・知的訴求）
- SNSでの共感訴求: 案A（感情的・体験的）
- イベント説明文: 案C（具体的なベネフィット）
- 思想的なポスト: 案D（哲学的・刺さる人に刺さる）

## 7. キーワード辞典（ブランドで使う言葉・使わない言葉）

### 積極的に使う言葉
```
✓ 感覚 / 五感 / 整う / 解放 / 儀式 / 朝 / 静寂
✓ リカバリー / リセット / 目覚め / 取り戻す
✓ 贅沢 / プレミアム / 本物 / 希少
✓ 身体 / 体感 / 温かい / 深い
✓ 仲間 / つながり / 人間 / 場所
```

### 使わない言葉（トーンを損なうもの）
```
✗ 参加者募集！/ 限定○名！/ お申し込みはお早めに！
✗ 健康になれます / ダイエット / カロリー
✗ ビジネス / 人脈 / 交流会 / ネットワーキング
✗ SNS映え / インスタ映え / バズる
✗ 効果 / 効能 / 実証済み
```

### 言い換えガイド
| 避けたい言葉 | HMCらしい言葉 |
|------------|-------------|
| 参加費 5,000円 | 一朝の投資 5,000円 |
| 定員20名 | 同じ朝を過ごす20人 |
| スマホ禁止 | スマホを置いてくる贅沢 |
| 健康に良い | 身体が喜ぶ |
| 交流できます | 名刺なしで話せる |
| 効果があります | そこにある体験だけが、証明する |
