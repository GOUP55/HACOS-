# HACOS LINE自動化 設計書
## LINE公式アカウント 自動返信・配信スケジュール完全設計

---

## 概要

LINEは HACOS の主なクライアント接点。  
自動化できる部分は自動化し、GOが人間的に関わるべき部分に集中する。

```
自動化する：定型的な通知・リマインダー・情報提供
GOが直接対応する：相談・SOS・個別フォロー・感情サポート
```

## 4. SOS対応プロトコル（自動＋手動の組み合わせ）

```
【フロー】

Step 1：クライアントが「SOS」「つらい」等を送信
          ↓
Step 2：自動返信が即座に「受け取りました、すぐ連絡します」を返送
          ↓
Step 3：GOのスマホに通知 → 15分以内に手動で返信
          ↓
Step 4：状況確認（「今どんな気持ち？教えてほしい」）
          ↓
Step 5：必要に応じて翌日セッション時間を前倒し

【GOの返信テンプレート（状況別）】

《食が崩れた》
「崩れたの教えてくれてありがとう。
 明日の朝、白湯から始めるだけでOK。
 それだけ。取り返そうとしないで😊」

《行けなくなりそう》
「どうした？何かあった？
 来られない理由を教えてほしい。
 日程調整できるよ。」

《やめたい気持ち》
「よく送ってくれたね。
 今、何がしんどい？
 聞かせてほしい。」
```

## 7. LINE公式アカウント 設定チェックリスト

```
  □ コホート・仲間・グループ

□ SOS系キーワードで通知音が変わるよう設定した（可能な場合）

□ Googleフォーム（週次チェックイン）のリンクを取得した
□ 予約ページのリンクを取得した
□ LPのリンクを取得した
□ Googleマップリンクを取得した

□ 配信スケジュールをカレンダーに登録した
  □ 月曜 08:00（目標宣言促し）
  □ 金曜 18:00（ベスト報告）
  □ 隔週水曜 12:00（測定週告知）
  □ 各種リマインダー
```

## 8. LINE運用 GOの週次タスク（10分/週）

```
月曜朝：
  □ グループに目標宣言を投稿する
  □ 週次チェックインフォームの回答を確認する
  □ 回答した全員に一言リプライする

金曜夜：
  □ グループにベスト報告を投稿する
  □ 個別トークに未読がないか確認する

測定週の後：
  □ 進捗レポートをLINEで送る（1人ずつ）
  □ 次回測定日をグループで確認する

随時：
  □ SOSには15分以内に返信する
  □ 食事写真が届いたら24時間以内にコメントする
```
