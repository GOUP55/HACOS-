## 1. 結論（現在地とギャップ、到達までの現実的な時間軸）

### 現在地（オーナー申告 2026-07-09）
- **現在の月商・利益率・会員数はオーナーから申告済み（具体的な数字は非公開・オーナーの手元メモで管理。帳簿での裏付けは未確認）**
- この現在地からの距離感：Phase 1（月商100万円）は既存商品のローンチ完了で狙える射程。月1000万円の利益（月商換算 約1,430万円）は現状から桁が変わる規模であり、§2-2のスケール商品なしには届かない
- 現地スタッフは2名が実在する（オーナー申告。名前は非公開・オーナー管理）。Phase 2の「対面キャパ拡大」は採用ゼロからではなく、この2名への委任設計から始められる（役割分担は `REMOTE_OPS_SYSTEM.md` §4・要オーナー決定）
- 確認できている事実（`BUSINESS_RULES.md`）：HMC朝活（定員10名＋追加枠3、¥2,000〜3,000/回）、体験パーソナル（¥3,500）、3ヶ月プレミアム（¥300,000・第1期定員3名・募集中、締切2026-07-31）、HACOSメンバーズ（準備中）、お弁当（¥1,300〜1,500/食）が既存商品。メンバーズ・21日間チャレンジ・弁当定期便はまだ「設計書はあるが未ローンチ」の状態（`NEXT_CHAT_HANDOFF.md`より）

### ギャップの構造的な事実
- 観音寺市の人口は約5.5万人。**対面・ローカル完結の事業だけで月1000万円の利益に到達するのは構造的に無理**（対面で回せる人数は、オーナー1人＋将来のスタッフ数名のカレンダーの上限に規定される）
- 月1000万円の「利益」を達成するには、利益率を仮に70%（仮定・要検証）とすると**月商換算で約1,430万円**が必要になる。これは既存のローカル商品（プレミアム・メンバーズ・HMC・弁当）の延長だけでは届かない規模であり、**地域を超えて売れるスケール商品（オンライン・デジタル・FC/ライセンス）が主力にならないと届かない**
- 「オーナーが海外にいても事業が回る」という前提を満たすには、収益を増やすだけでなく、**運営そのものをオーナーの現地稼働から切り離す**（スタッフ／コーチへの委任、コンテンツの資産化、自動化された配信・決済・サポート）ことが同時に必要

### 現実的な時間軸（仮定）
| フェーズ | 目標月商 | 目安の利益 | 到達時期の目安（仮） | 前提 |
|---|---|---|---|---|
| Phase 1 | 約100万円 | 約50〜70万円 | 3〜6ヶ月後（〜2027年初頭） | 既存ローカル商品（プレミアム・メンバーズ・HMC・弁当・チャレンジ）の完成・ローンチ |
| Phase 2 | 約300万円 | 約150〜200万円 | 12〜18ヶ月後（2027年後半〜2028年） | スタッフ採用による対面キャパ拡大＋オンライン商品の立ち上げ |
| Phase 3 | 約1,000万円台の利益を目指す | 約1,000万円（目標） | 2〜3年後以降（仮・実行力次第で前後） | オンライン会員の大規模化・デジタル商品・FC/ライセンスが収益の主力になる |

**正直な結論：** 月1000万円の利益は「今の事業の延長線上」にはなく、**新しい収益の柱（特にオンライン・デジタル・FC）をゼロから立ち上げ、それが数百〜千名規模の顧客に育つこと**が前提になる。これは早くて2〜3年、集客・商品開発・採用のいずれかが計画通りに進まなければさらに長くかかる。段階を飛ばして一気に到達する近道はない。

## 4. オーナー不在耐性（不在でも売れる／回る度合い）

| 商品・柱 | 不在耐性 | 理由 |
|---|---|---|
| HMC朝活 | 低 | オーナー本人が指導する対面クラス。スタッフトレーナーへの委任で「中」まで上げられるが現状は本人依存 |
| 体験パーソナル | 低 | 対面のトレーナー稼働そのもの。予約受付は自動化済みだが、実施は人が必要 |
| 3ヶ月プレミアム（現行・対面） | 低 | 対面パーソナル＋食サポート＋クロージング面談がオーナー本人前提。高単価・高信頼ゆえに委任のハードルも高い |
| HACOSメンバーズ（対面） | 低〜中 | 対面パーソナル部分は低いが、LINEサポート部分はスタッフでも運用可能。段階的に中へ引き上げられる |
| お弁当定期便 | 中 | 調理・受け渡しはスタッフに委任可能。メニュー監修・仕入れ判断は当面オーナーが必要 |
| 21日間チャレンジ | 中 | 配信文面は事前作成済みなら自動配信可能。グループ振り返りのファシリは人手が必要（オーナーでなくてもよい） |
| パーソナルブランド発信（Instagram・note・Voicy） | 中 | コンテンツ制作自体はオーナー本人の顔・声が必要。ただし公開済みコンテンツは不在中も閲覧・リード獲得を生み続ける「資産」になる。新規発信が止まると徐々に減衰する点は要注意 |
| HACOSオンラインメンバーズ（新設） | 高 | 動画教材＋テンプレ化したLINEサポート＋委任コーチ運用にすれば不在耐性は高い。ただし立ち上げ初期の教材制作・監修はオーナーが必要 |
| デジタル商品（note有料記事・電子書籍） | 高 | 一度作れば販売・提供は自動。更新頻度が落ちても収益は継続する「積み上げ型」資産 |
| 法人向け健康経営プログラム（新設） | 中 | 初期営業・契約はオーナーブランドの信頼が必要だが、実施自体はスタッフ・オンラインプログラムへの委任が可能 |
| ライセンス／FC（新設） | 高 | 一度カリキュラムを体系化・認定すれば、ロイヤリティ収益はライセンシー側の活動から生まれる。ただし教育・品質管理の仕組み構築という初期投資が必要 |

**読み取り方：** Phase1〜2で稼ぐ柱（対面）は不在耐性が低い。Phase3で主力にするべき柱（オンラインメンバーズ・デジタル商品・FC）は不在耐性が高い。**「不在でも回る事業」を実現する道筋は、Phase1〜2の対面収益で資金と信用を作りながら、並行してPhase3の高耐性商品をゼロから育てておくこと**にある。Phase3に入ってから作り始めるのでは遅い。
