
## #1 PT回数を「週1回→週2回」に引き上げる ★★★★★

### 現状の問題

```
業界相場（2026年）：
  PT月4回（週1回）の相場 ＝ 月¥2〜5万（通常の月額制ジム水準）
  PT月8回（週2回）の相場 ＝ 月¥6〜10万（プレミアム・ダイエット特化）

つまり現状のHACOSプランは：
  「¥10万払っているのに、週1回しか来ない」という設計になっている。

リスク：
  週1回では筋肉への刺激が不十分で脂肪燃焼が鈍い。
  成果が出にくい → 口コミが生まれない → 客単価を守れない。
```

### 改善案：運動を2段階設計に変える

```
【スタンダード運動】¥55,000（現行維持）
  PT月4回（週1回）
  グループ月2回
  朝ラン参加
  動画配信
  ← 「習慣化・入門」として再ポジション

【プレミアム運動】¥75,000（新設）
  PT月8回（週2回）← これが本来の「本気ダイエット」水準
  グループ月2回
  朝ラン参加
  動画配信
  ← 「本気・結果保証」として打ち出す
```

### 再設計後の全体プラン

```
プランA（入門）  ¥100,000/月
  スタンダード運動¥55,000 + 食¥45,000

プランB（本気）  ¥120,000/月  ← メイン推奨
  プレミアム運動¥75,000 + 食¥45,000
  ※1セッション¥9,375（相場¥11,000〜¥18,600を大きく下回るお得さ）

プランC（最速）  ¥130,000/月
  プレミアム運動¥75,000 + 食¥45,000 + 結果加速オプション¥10,000
```

### 価格根拠（クライアントへの説明文）

```
「PT月8回（週2回）を他のジムで受けると、
 1回¥11,000〜¥18,600が相場です。
 8回で最低¥88,000〜¥148,000。
 HACOSのプランBは月¥120,000で
 PT8回＋食事指導＋グループ＋朝ラン＋動画がすべて入っています。」
```

## #2 オプション価格と内容を全面見直す ★★★★☆

### 現状の問題

```
現在のオプション：
  習慣化サポートパック ¥7,000（LINE4回 + 朝活4回）
  結果加速パック ¥10,000（LINE4回 + セミパ4回）

問題1：セミパ4回で¥3,000追加は安すぎる
  セミパ1回あたり¥750という計算になり、サービスの価値を自ら下げている。

問題2：¥100,000に対して¥7,000〜¥10,000では
  「あってもなくてもいい」という心理になる。

問題3：「LINE相談」という言葉が弱い
  LINEはすでに基本プランに含まれているはずで、
  「相談4回」という制限設定がむしろ不安を生む。
```

### 改善案：オプションを「価値で売る」設計に変える

```
【習慣化サポートパック】¥7,000 → ¥12,000
  ├ 朝活参加4回（¥3,000相当×4 = ¥12,000の価値）
  ├ 週次モチベーションセッション（LINE音声5分×4回）
  └ 「習慣トラッカー手帳」プレゼント（¥1,500相当）
  
  訴求：「続けるための仕組みを買う」
  通常¥15,500相当 → ¥12,000

【結果加速パック】¥10,000 → ¥20,000
  ├ セミパーソナル追加4回（¥4,000相当×4 = ¥16,000の価値）
  ├ 体組成測定 追加1回（¥2,000相当）
  └ 栄養士LINEフォロー週1回（¥2,000相当）
  
  訴求：「成果のスピードを2倍にする」
  通常¥20,000相当 → ¥20,000（同額だが内容で納得感を作る）

【フルサポートセット】両方セット
  通常¥27,000 → ¥25,000（¥2,000 off）
  
  訴求：「完全にお任せしたい人向け」
```

### 価格感の心理設計

```
¥100,000 ベース  ← 「十分な価値」
+ ¥12,000       ← 「たったコーヒー代4杯分」
+ ¥20,000       ← 「1回のPT追加よりお得」
+ ¥25,000       ← 「全部で¥125,000。外のジムでPT10回分」

→ オプションの「追加感」より「当然感」を演出する。
```

## #3 「食¥45,000」の中身を完全に可視化・再設計する ★★★★☆

### 現状の問題

```
相場データ（2026年）：
  食事指導のみのオプション相場 ＝ 月¥30,000〜¥50,000
  写真チェック + アドバイスのみ ＝ 月¥20,000〜¥30,000
  栄養士カウンセリング（月1回60分）＝ ¥5,000〜¥10,000単体

現状のHACSの食¥45,000に含まれるものが不明確。
→ 高いのか安いのか、クライアントが判断できない。
→ 「食事指導に4.5万も払うの？」という疑念が生まれやすい。
```

### 改善案：食サポートの「内訳×相場比較」を明示する

```
【HACOS 食サポートパッケージ ¥45,000の内訳】

  栄養士カウンセリング × 1回（60分）   相場¥8,000  → ¥0（込み）
  週次ミールプラン（4週分）             相場¥4,000  → ¥0（込み）
  発酵食ワークショップ × 1回（60分）   相場¥5,000  → ¥0（込み）
  食事写真フィードバック（週1回）       相場¥8,000  → ¥0（込み）
  コンビニ・外食ガイドカード            相場¥1,500  → ¥0（込み）
  ──────────────────────────────────────
  合計相場：¥26,500〜¥35,000
  
  それに加えてHACOSだけの独自価値：
  ├ 発酵食レシピ（讃岐食材ベース）
  ├ 「食と動の連動プログラム」設計
  └ LINEでの食事相談（都度対応）
  
  → ¥45,000は「割安」どころか「圧倒的にお得」という事実を伝える。
```

### 「食の価値」の見せ方を変える

```
改善前の表現：
  「食 ¥45,000」

改善後の表現：
  「HACOS 食サポートプログラム ¥45,000
   （栄養士指導 + 週次プラン + 発酵食ワークショップ + 毎週フィードバック込み）
   他社では別途¥30,000〜¥50,000かかるサービスが含まれています。」
```

### 食の「Lite版」を新設してプランAを守る

```
現状のプランAの問題：
  ¥100,000のうち¥45,000が食 → 食サポートに高い印象

改善案：
  「食 Lite版」¥30,000 を新設

  食 Lite版（¥30,000）：週次ミールプラン + 食事写真フィードバック
  食 Full版（¥45,000）：Lite + 栄養士カウンセリング + 発酵食ワークショップ

→ プランA（入門）を¥85,000に下げることができ、
  入口を広げながら食フル版へのアップグレードを自然に誘導できる。
```
