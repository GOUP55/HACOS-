# ダイエットプラン 改善提案 TOP3

> 現状プランの課題と、具体的な改善アクションをまとめました。

---

## 改善点 TOP3 の全体マップ

```
現プランの弱点
│
├── #1 体組成データの「見える化」が弱い
│       → 感覚頼りの進捗管理になりがち
│
├── #2 食事サポートが「提案止まり」で実行支援がない
│       → 知識は得ても行動に落とし込めない
│
└── #3 プラン終了後の「継続設計」がない
        → 卒業後にリバウンド・離脱するリスク大
```

## #1 体組成データの「見える化」強化

### 現状の課題
- 月1回の体組成測定だけでは変化を感じにくい期間がある
- 測定結果を渡して終わりになりやすい（数字の意味が伝わらない）
- モチベーション低下時のデータ的サポートが薄い

### 改善案

#### A. 測定頻度を上げる（月1 → 隔週1回）
```
改善前：月1回（3ヶ月で3回）
改善後：隔週1回（3ヶ月で6回）

追加コスト：+¥2,000 × 3回 = +¥6,000
効果：変化の「手触り」が増え、モチベーション持続
```

#### B. 進捗レポートの視覚化
毎回の測定後に「1枚紙のビジュアルレポート」を手渡し。

```
【HACOS 進捗レポート】

Week __ /__  　　　__/__〜__/__

体重：__._ kg（先週比 ±_._ kg）▼▼▼▼▼▼▼▼▼▼▼▼▲▲▲▲
体脂肪率：__._% （先週比 ±_._）
筋肉量：__._kg

今週のMVP行動：_______________________
来週の1つの約束：______________________

トレーナーコメント：
```

#### C. LINEでの週次自己チェックイン
毎週月曜に自動送信される「3問チェックイン」を導入。

```
今週の3問チェックイン
Q1：今週の睡眠の質は？（1〜5）
Q2：食事の自己評価は？（1〜5）
Q3：今週のベストな選択は何でしたか？
```

**実装コスト：** Google フォーム + LINE公式アカウントの自動配信設定（初期1時間）

## #2 食事サポートを「計画提供」から「実行支援」へ

### 現状の課題
- 週次ミールプランを渡しても、買い物・調理が続かない
- 外食・コンビニでの選択基準を知らないまま
- 食事ログが続かず、途中で脱落しやすい

### 改善案

#### A. 月1回「HACOS キッチン実習」を追加
```
内容：60分の料理ワークショップ（少人数 2〜4名）
場所：HACOSキッチン（or コミュニティパートナーのキッチン）
内容例：
  - タンパク質を効率よく摂れる3品
  - 作り置きオカズ × 2種
  - 翌週ミールプランの食材リスト手渡し

追加料金：+¥3,000/回（月1回 × 3ヶ月 = +¥9,000）
```

#### B. コンビニ・外食ガイドの作成
1枚でわかる「外でも困らないHACOS食ガイド」をLINEで送付。

```
【HACOS 外食・コンビニガイド（携帯版）】

◎ セブン：サラダチキン + 卵 + 野菜スープ（タンパク質35g）
◎ ファミマ：豆腐バー + ゆで卵 + 野菜ジュース
◎ 定食屋：焼き魚定食（ご飯 少なめ依頼、味噌汁は椀半分）
◎ ランチ：主食より先にサラダ・汁物を食べる（血糖値スパイク防止）

× 避けたいもの：揚げ物定食、甘い缶コーヒー、菓子パン
```

#### C. 食事ログの「写真1枚ルール」定着化
毎食の写真をLINEで送るだけ。トレーナーが週1回まとめてフィードバック。

```
改善前：食事日記（テキスト）→ 続かない
改善後：写真 + 一言（「ランチ」でOK）→ 続く

週末フィードバック例：
「今週は朝食のタンパク質が少なめでした。
 卵1個 or プロテインを追加してみましょう！
 外食のサラダ最初食べ、完璧でした！」
```

## #3 「卒業後」の継続設計（リバウンド防止）

### 現状の課題
- 90日間で終わり → 翌月からひとり → リバウンド・離脱
- 卒業後の受け皿がない（次のプランへの導線が弱い）
- せっかく築いたコミュニティとのつながりが切れる

### 改善案

#### A. 「HACOS アフターケアメンバーシップ」設計
```
月額 ¥15,000〜¥20,000 の継続プラン

内容：
  - PT月2回（隔週）
  - 体組成測定 月1回
  - LINEサポート継続
  - セミパーソナル参加 無制限
  - HMC参加 月1回

卒業時に「3ヶ月先払い割引」でオファー：
  通常 ¥15,000 × 3ヶ月 = ¥45,000
  → 先払い ¥40,000（¥5,000 off）
```

#### B. 「卒業生コミュニティ」ネットワーク化
```
LINE オープンチャット「HACOS ダイエット卒業生」

目的：
  - 仲間意識による継続動機づけ
  - 食レシピ・トレーニング体験のシェア
  - 月1回の集合イベント（ハイキング・食事会など）

HACOSにとってのメリット：
  - 既存顧客のLTV（顧客生涯価値）向上
  - 卒業生からの紹介でコスト低い新規獲得
  - コミュニティ事業の活性化
```

#### C. 90日「卒業セッション」を儀式化
```
最終週に「成果発表＆次の目標設定」セッション（60分）を実施。

内容：
  1. 3ヶ月のBefore/After 体組成レポート共有
  2. 「一番変わったこと」を本人の言葉で語る
  3. 次の90日の目標を1枚紙に書く
  4. HACOS継続プランの案内

効果：
  - プランへの満足度・完走感が上がる
  - 自然にアフターケアの話ができる
  - SNS投稿のネタになる（口コミ獲得）
```
