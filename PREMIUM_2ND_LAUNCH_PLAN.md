## 3. 第1期実績の活用方法

**大原則: 実測のみ・捏造禁止。数値は必ず計測時期を明記し、断定表現を避ける。**

| 項目 | 方針 |
|---|---|
| 取得タイミング | 体組成測定は月次実施（premium.html記載の「体組成測定＋月次フィードバックレポート」）のため、1ヶ月目終了時点（8月上旬頃）に中間データが取得できる見込み。声はHMC参加時・カウンセリング時に自然な会話の中でヒアリング |
| 許諾の取り方 | 口頭確認＋LINEでのテキスト同意（「この内容を第2期の説明で紹介してもいいですか」と明示的に確認）。書面の同意書がある場合はそちらを優先 |
| 掲載の任意性 | premium.html §8.5の特典記載と同様、「掲載は任意・断っても受講内容や料金に一切影響しない」ことを本人に明言する |
| 数値の扱い | 「◯ヶ月目時点で◯kg」のように**計測時期を必ず併記**。3ヶ月終了前の数値を「3ヶ月の実績」であるかのように見せない。「効果には個人差があります」を必ず付す |
| 声の扱い | 個人が特定されない範囲での言い換えを基本とし、本人の言葉をそのままコピーする場合も掲載前に本人確認を取る |
| 使わない情報 | 許諾が取れていない実測値・写真・氏名は一切使用しない。定員3名という少人数ゆえ、匿名化しても個人特定されやすい点に留意し、本人の意向を最優先する |

## 4. 価格の検討メモ（¥300,000維持 vs ¥350,000正規化）

premium.htmlの現行表示は「通常¥350,000（3ヶ月）→ 先着3名限定 ¥300,000」。第2期でこの値引き構成を続けるか、正規化するかは**第1期の成約状況を見て判断する分岐**として整理する。最終決定はオーナー。

| 判断材料 | ¥300,000維持が妥当なケース | ¥350,000正規化が妥当なケース |
|---|---|---|
| 第1期の成約スピード | 締切ギリギリまで埋まらなかった／値引きが後押しになった様子があった | 早期に定員が埋まった／価格が障壁になっていなかった |
| 第1期生からのフィードバック | 「¥300,000でも高いと感じた」という声があった | 「むしろ安すぎるのでは」という声があった、または特に価格への言及がなかった |
| 提供コストの変化 | 変化なし、またはコスト増がない | トレーナー稼働・食サポート原価が増え、¥300,000では採算が厳しい |
| 限定性としての効果 | 「先着3名限定・値引き」の訴求が予約行動に効いていた（GA4のCTAクリック傾向等で確認） | 値引きよりも「少人数制・伴走」自体が動機になっていた |
| 市場との比較 | 通常パーソナル24回¥150,000〜との差が大きすぎて心理的ハードルが高かった | 差額に見合う価値（食サポート・コミュニティ込み）が十分伝わっていた |

**分岐の考え方（案）**:
- 第1期が定員3名を締切前に確保 → ¥350,000正規化を検討する余地あり（値引きなしでも需要がある可能性）
- 第1期が締切ギリギリ・値引きが決め手になった様子 → ¥300,000維持、または別軸の限定性（早期申込特典など）で調整
- いずれの場合も、第1期生との価格差（既存生が¥300,000で、新規が¥350,000）が生じる場合は、既存生への説明・不公平感のケアを事前に検討する
