## #7 15〜20万円の価格に見合う「体験品質（UX）」設計

### 現状の課題
プランの中身は良くても、**体験の「見た目と手触り」が普通のジムと変わらない。**

15〜20万円を払うということは、クライアントは「特別な体験」を期待している。  
しかし今のフローは：
```
カウンセリング → 測定 → LINE登録 → Week1スタート
```
これでは高級レストランに行って「紙皿で出てきた」ような落差が生まれる。

体験品質が低いと何が起きるか：
- 「この値段でこれか」という心理的不満が蓄積する
- 途中離脱時の「もったいない感」が薄い（だから辞めやすい）
- 口コミで「内容はよかったけど…」という評価になる

### 改善案

#### A. 「入会ウェルカムキット」を作る
初日に渡す箱（A4サイズ・黒箱）の中身：

```
【HACOS ウェルカムキット】

1. ウェルカムレター（GOの直筆署名入り・1枚）
   「あなたのこの決断を、全力で支えます」

2. HACOS 専用プランブック（A5・リング綴じ）
   ・自分の目標・体組成初期値の記入欄
   ・12週間のロードマップ（記入していくタイプ）
   ・週次チェックイン欄（手で書く体験）

3. HACOS オリジナルシェイカーボトル（マットブラック）
   「HACOS LIFESTYLES」刻印入り

4. 初週のミールプラン（印刷・A4・ラミネート）

5. 「90日後の私へ」カード
   スタート時に自分で書いて封筒に入れる。
   卒業セッションで開封する。
```

**コスト：** 約¥3,000〜¥5,000/キット（大量発注で圧縮可）  
**効果：** 「本気のプランに入った」という自己認識が継続意欲を高める

---

#### B. セッション空間の「五感設計」を統一する
```
視覚：セッション前にHACOS専用の照明モードに切り替え
      （鍛錬の間：強め白色　帰還の縁側：暖色・間接照明）

嗅覚：開始時にアロマディフューザー ON
      （ユーカリ or ヒノキ → 覚醒・集中）
      終了時：ラベンダー系に切替（回復モード）

聴覚：セッション用プレイリストを固定
      （Spotifyで「HACOS Session」プレイリストを公開）

触覚：タオルをHACOSブランドに統一
      終了後にホットタオル提供（1枚¥30のコスト・体感価値は高い）
```

**効果：** 「HACOSに来ると整う」という感覚記憶が蓄積する。  
感覚が統一されるほど、空間への依存（＝継続）が生まれる。

---

#### C. SNSで「映える」進捗体験を設計する
```
Week 4・8・12 の測定日を「記念日」として演出：

① 体組成測定 → 結果をカードに印刷して手渡し
② 「今日の一言」をボードに書いて写真撮影
   （例：「4週間で -2.3kg、今日が一番元気！」）
③ ビフォーアフター数字カード（顔なし・数字だけ）を
   インスタストーリーズ用テンプレで提供

→ クライアントが「シェアしたくなる」体験を設計する
  （強制しない。したくなる仕掛け）
```

## #8 「-5kg」という数字目標の脆さ ── 動機を人生レベルに深堀りする

### 現状の課題
今のプランの目標設定は：
```
体重：___ kg → ___ kg（-___ kg）
体脂肪率：___% → ___%
```
これだけ。

数字目標の致命的な弱点：
- **停滞期（3〜4週目）に「-0.1kgしか減ってない」で心が折れる**
- 数字は「結果」であり「動機」にはならない
- 日常のしんどい場面で「-5kg のために今日我慢しよう」とはならない

行動科学の知見：  
> 「体重を減らしたい人」より「健康的な人として生きたい人」の方が、  
> ダイエットに長期的に成功する確率が2〜3倍高い。

### 改善案

#### A. 初回カウンセリングに「5つのなぜ」を導入する
```
初回カウンセリング（30分 → 45分に延長）の後半15分：

Q1「なぜダイエットしたいですか？」
→「痩せたいから」

Q2「なぜ痩せたいですか？」
→「自信を持てるようになりたい」

Q3「なぜ自信を持ちたいですか？」
→「仕事でもっと堂々と話せるようになりたい」

Q4「なぜ堂々と話せるようになりたいですか？」
→「部下に信頼されるリーダーになりたい」

Q5「それが実現したとき、あなたはどんな人ですか？」
→「チームに安心感を与えられる、体も心も整った人」

← ここが「本当の目標」
```

この会話で出てきた言葉を、プランブックの表紙に書く。  
**「体重 -5kg」ではなく「チームを支えられるリーダーになる」**が目標になる。

---

#### B. 「なりたい自分宣言カード」をプランブックの冒頭に置く
```
【私の90日間の約束】

私は ______________________ な人間です。（現在形で書く）

その姿に近づくために、私は今日から：
  ・毎週2回、HACOSに来る
  ・食事の写真を撮る
  ・水を1日2リットル飲む

この約束を、GOトレーナーと交わします。

署名：____________　　日付：____年__月__日
```

**なぜ現在形か：**  
「なりたい」ではなく「私はそういう人間だ」と書くことで、  
行動が「目標達成のための我慢」ではなく「自分らしい選択」に変わる。

---

#### C. 停滞期に備えた「動機の予防接種」
```
Week 1 のセッション終わりに渡す封筒：

「4週目に開けてください」と書いてある

中身：
  「いまあなたは停滞期にいるかもしれません。
   でも思い出してください。
   あなたが最初に教えてくれた言葉：
   『___________________________』（初回の5つのなぜで出た本質動機）
   体重の数字は、あなたの全てではありません。
   明日もここに来てください。
   ── GO」
```

**効果：**  
停滞期に「それでも来る理由」が手紙という形で残っている。  
「トレーナーが4週目のことを考えてくれていた」という体験が信頼になる。

## #9 「食と動と人を繋ぐ」HACOS哲学を食事プランに本気で反映させる

### 現状の課題
HACOSのコンセプトは：
> **「食と動、人と大地が繋がる場所」**

しかし、現在の食事プランは：
- タンパク質○○g/日
- カロリー収支の管理
- コンビニの選び方

**これは他のどのダイエットプランにもある内容。HACOSである必要がない。**

さらに言えば、HACOSはすでに以下を持っている：
- 塩糀・醤油糀・酵素シロップなどの発酵食品開発候補
- 讃岐（香川）の食文化・地元食材へのアクセス
- 「土と繋がる食」という確固たる哲学

これをプランに本気で組み込めば、**全国のどのダイエットプログラムにも真似できない独自性**が生まれる。

### 改善案

#### A. 「HACOS 食哲学」を3原則として言語化し、プランの軸にする
```
HACOS ダイエット 3原則

原則1：「引く食」より「足す食」
  カロリー制限ではなく、良いものを足していく発想。
  タンパク質・発酵食品・野菜を足すと、自然に悪いものが減る。

原則2：「土に近いものを食べる」
  加工度の低いもの、地元で作られたもの、季節のものを優先。
  讃岐の食材（瀬戸内の魚・地元野菜・讃岐米）を積極的に使う。

原則3：「食べる行為を整える」
  何を食べるかより、どう食べるか。
  よく噛む・感謝する・一人で食べない──食の質は習慣の質。
```

**この3原則を印刷してプランブックの見開きに入れる。**  
カロリー表ではなく哲学が最初に来ることで、プランの格が変わる。

---

#### B. 「HACOS 発酵食プログラム」を食事サポートの核にする
```
月1回のキッチン実習を「発酵食ワークショップ」に格上げ：

Month 1「塩糀を仕込む」
  ・塩糀の作り方（10分）
  ・塩糀漬けチキンの調理（20分）
  ・「塩糀があれば外食を3回分置き換えられる」解説

Month 2「味噌汁を再設計する」
  ・だしの取り方（煮干し・昆布）
  ・腸内環境と発酵の関係
  ・毎朝の味噌汁がダイエットに効く理由

Month 3「酵素シロップを作る」
  ・旬のフルーツで発酵ドリンク
  ・砂糖への依存から抜ける味覚リセット
  ・卒業後も「自分で作れる体」

→ クライアントは技術を持ち帰る。食の自立が起きる。
```

**副次的な価値：**  
- これが「HACSのダイエットは発酵食が核」というSNSコンテンツになる
- 「塩糀・酵素シロップ」の物販展開への入口になる（将来の収益柱）
- 「讃岐の食文化を守る」というストーリーがメディア露出に繋がる

---

#### C. 地元食材との連携で「食の体験」を唯一無二にする
```
現状：スーパーで買える食材でミールプランを設計
　↓
改善：地元農家・漁師と連携した「HACOS食材ボックス」

月1回、地元の旬の食材セット（約¥3,000相当）を
プランに組み込む（または別途¥2,000で提供）：

例）
・丸亀の瀬戸内鮮魚（アジ・サワラ）
・地元農家の季節野菜（なす・ししとう）
・讃岐産の発芽玄米 or もち麦
・今月の発酵食（塩糀 or 甘酒）

付属：「今月の食材でできる3品レシピカード」

効果：
  ・食べることが「楽しみ」になる（義務感からの脱却）
  ・地元農家との連携がHACOSのストーリーになる
  ・「丸亀で作られた体」という独自ポジションが生まれる
```
