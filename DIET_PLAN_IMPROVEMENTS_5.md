## #13 3つのペルソナが全員「同じプラン」を受けている

### 現状の課題
プランには3つのターゲットペルソナが定義されている：

```
Aさん（35歳女性・会社員）：産後太り・挫折経験多い・管理してほしい
Bさん（42歳男性・経営者）：健診引っかかり・忙しい・効率重視
Cさん（28歳女性・フリーランス）：見た目改善・仲間と頑張りたい
```

しかし渡すプランも、セッション時間も、コミュニケーション頻度も全員同じ。  
これはアパレルで「サイズM一択」を全員に売るのと同じ問題。

ペルソナごとのリアルな差：

| 比較項目 | Aさん（会社員ママ） | Bさん（経営者） | Cさん（フリーランス） |
|---------|-----------------|--------------|-------------------|
| 使える時間 | 朝7時前 or 夜9時以降 | 早朝か土曜のみ | 午前中が自由 |
| 最大の障壁 | 育児・家事との両立 | 出張・接待 | 自己管理の甘さ |
| 動機の核 | 「子供のそばで元気でいたい」 | 「健康リスクの排除」 | 「自己肯定感の回復」 |
| 反応する言葉 | 「ムリなく続けられる」 | 「効率的に・確実に」 | 「一緒に・楽しく」 |
| サポート頻度 | 毎日短く | 週1まとめて | 毎日双方向で |

### 改善案

#### A. 3つの「ライフスタイルパッケージ」に分岐させる
```
HACOS LIFE ─── 産後・子育て世代向け
  ・早朝・夜間帯のセッション枠を確保
  ・「10分でできる宅トレ」動画をLINEで週1配信
  ・ミールプランは「子供と一緒に食べられる料理」優先
  ・キーワード：「ムリなく・家族ごと・長く」

HACOS EXEC ─── 忙しい経営者・管理職向け
  ・土曜 or 早朝6時台のセッション固定
  ・出張時の「ホテルでできるトレーニング動画」提供
  ・外食・接待での選択ガイドを手帳サイズで携帯できる形に
  ・キーワード：「効率・数値・ROI」

HACOS FREE ─── 20〜30代フリーランス・自由業向け
  ・コホートモデル（仲間との同期）をこのプランの核にする
  ・SNSでの発信を一緒に設計（自己ブランド化のサポート）
  ・カフェワーク時間に合わせた午前セッション
  ・キーワード：「仲間・楽しさ・自己表現」
```

**実装コスト：**  
コンテンツの中身は共通90%。見せ方・タイトル・タイミングを変えるだけ。  
3種類のランディングページ（もしくはInstagramストーリーズのバリエーション）で展開。

#### B. カウンセリングの最初に「ライフスタイルスコア」を測定する
```
5問のスクリーニング（3分）：

Q1「平日の自由時間は合計何時間ありますか？」（1h以下 / 1〜2h / 3h以上）
Q2「週に何回、外食または接待がありますか？」
Q3「一番近い人（家族・パートナー）は、あなたの挑戦を応援していますか？」
Q4「これまでのダイエット挫折の一番の理由は何でしたか？」
Q5「ひとりで頑張りたい vs 仲間と一緒に頑張りたい」（5段階）

→ 結果から「HACOS LIFE / EXEC / FREE」を判定して提案。
  「あなたにはこのタイプが合います」という提案は購入決断を後押しする。
```

## #14 年末年始・夏・飲み会シーズンへの「崩れない設計」がない

### 現状の課題
90日プランの中に、必ず以下の「計画キラー」がある：

```
1〜3月スタート：年度末・歓送迎会シーズンに入る
4〜6月スタート：梅雨・GW明けのモチベーション低下期
7〜9月スタート：猛暑・夏休み・お盆帰省
10〜12月スタート：忘年会・クリスマス・正月
```

どの月にスタートしても、90日の中に必ず「崩れやすい週」が入る。  
しかし今のプランにはその対策が一切ない。

研究では：
> ダイエット離脱の42%はライフイベントの「崩れ」がきっかけで起きる。  
> 逆に言えば、この崩れに事前対策できれば42%の離脱を防げる。

### 改善案

#### A. スタート時に「90日の崩れ予測カレンダー」を作る
```
初回カウンセリングで一緒に記入するシート：

【私の90日 危険ゾーンMAP】

Week __：__月__日 → （忘年会 / 出張 / 帰省 / その他：＿＿）
Week __：__月__日 → （          〃         ）

合計 __ 個の危険ゾーンがあります。
これを「事前に知っている」だけで、乗り越えられる可能性が大幅に上がります。

→ 各危険ゾーンに「その週の特別ルール」を事前設定する。
  例）「忘年会の週は、食事ログだけ続ける。トレーニングはお休みOK。」
```

#### B. 「崩れた日のリカバリープロトコル（3ステップ）」を配布する
```
食べすぎた・飲みすぎた・運動できなかった日の翌朝：

Step 1（翌朝）：水500ml + 白湯500ml。食事は普通に食べる。
  NG：「昨日の分を断食で取り返そう」→ 逆効果

Step 2（翌日のトレーニング）：強度を70%に落とす。
  「来た」だけで◎。完璧にやろうとしない。

Step 3（LINEでGOに報告）：
  「昨日崩れました」と送るだけ。
  返信：「OK、今日からまたスタートです。」

ポイント：「崩れた＝失敗」ではなく「崩れた後の行動が全て」という
         マインドセットをプランの文化として植え付ける。
```

#### C. 「崩れOK週」を最初から1〜2週間プランに組み込む
```
12週間のロードマップを「10週間の本気 + 2週間の余白」として設計する。

Week 1〜10：プログラム通りに進む
Week 11〜12：「余白週」── 崩れた週の回収、または目標上方修正

効果：
  ・完璧主義からの解放（「2週間の余白があるから大丈夫」）
  ・実際に全員が10週で終わる → 2週は「ボーナス期間」になる
  ・ボーナス期間の追加成果がSNS映えするコンテンツになる

クライアントへの伝え方：
  「12週間のうち2週間は"調整週"です。
   人生は計画通りにいきません。
   その前提で設計された、本物のプログラムです。」
```

## #15 「なぜGOがやるか」がプランに一切入っていない

### 現状の課題
15〜20万円という高価格帯で人が買う決め手は何か。  
それは「この人に任せたい」という**人への信頼**だ。

しかし今のプランには、GOという人間が一切見えない。

```
現在見えているもの：
  ✅ プログラムの内容
  ✅ 料金
  ✅ 期間
  ✅ 成果の数字目標

見えていないもの：
  ❌ なぜGOはこの仕事をしているのか
  ❌ GOが乗り越えてきたこと（挫折・転換点）
  ❌ GOがクライアントに本当に伝えたいこと
  ❌ GOがどんな人間か（価値観・生き方）
```

競合のジムもプログラムの「内容」は売っている。  
**GOの「物語」を持っているのはHACOSだけ。**

### 改善案

#### A. GOのストーリーを「1枚紙のWelcomeレター」として形にする
```
ウェルカムキットに入れる直筆署名レターの内容骨子：

「なぜ私がこのプランを作ったか」

・（自分自身の転換点：挫折 or 発見 or 決断）
・「食と動が繋がったとき、人生が変わった」という実体験
・観音寺・香川でこれをやる理由（地元への思い）
・あなたに伝えたいたった1つのこと

文字数：400字以内。手書き or 手書き風フォント。
```

#### B. InstagramとLINEで「GOの日常」を見せるコンテンツを作る
```
クライアントがプランを検討するとき、必ずSNSを確認する。
そこで「GO自身がどう生きているか」が見えることが、
最後の「この人に任せよう」という決断を生む。

週1本：「今日のGO」シリーズ
  ・GO自身のトレーニング動画（30秒）
  ・GO自身の食事（「今日の発酵食」1枚）
  ・GOの考え（「今週感じたこと」140字）

月1本：「GOの生き方」シリーズ
  ・なぜ観音寺でやるのか
  ・大切にしていること
  ・影響を受けた人・本・体験
```

#### C. プランブックの冒頭に「GOとの約束」ページを設ける
```
プランブック 2ページ目：

【GO から あなたへの約束】

1. あなたの90日間を、私は本気で一緒に戦います。
2. 結果が出ない原因を、一緒に考え続けます。
3. あなたの「なぜ」を、私は忘れません。
4. 食と動と整えることが、人生を変えると信じています。

── GO

【あなたから GO への約束】

私は _____________________ のためにここに来ました。
この90日間、正直に向き合います。

署名：_______________　 日付：____年__月__日
```

**これが「契約書」ではなく「誓い」になる。**  
プランが「サービスの購入」から「人と人の関係」に変わる瞬間。
