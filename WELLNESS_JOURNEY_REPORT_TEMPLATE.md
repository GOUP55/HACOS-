## 紙面構成（A4縦・上下2分割）

### 上半分: あなたの7日間（変化の見える化）

| 要素 | 内容 |
|---|---|
| ヘッダー | 「◯◯さんの Wellness Journey Report」＋期間（8/1〜8/9等） |
| 測定の変化 | Day0→Day6: 体重／体脂肪率／ウエスト（任意）※本人にのみ渡す・比較競争にしない |
| 行動の記録 | 朝RUN参加 __/6回・グルトレ __/2回・宿題実施 __/7日・食事写真 __回 → **数値が動かなくても「行動の変化」を必ず褒める欄** |
| 主観の変化 | 睡眠の質・疲労感・気分（Day0とDay6の10点満点比較） |
| トレーナーからの一言 | GO直筆または手書き風。その人だけの言葉で（テンプレ文の使い回し禁止） |

### 下半分: 次の30日でやること（＝30DAYSの提案書）

| 要素 | 内容 |
|---|---|
| あなたに合っていた習慣 TOP3 | 7日間で効いたもの（例: 朝の30分／夕食の整え／寝る前ローラー） |
| 次の30日の目標 | 本人がDay6に自分の言葉で書く欄（書いてもらうことが申込動機になる） |
| 30DAYSのご案内 | 週2グルトレ×8回＋HMC朝活＋食事サポート¥20,000相当＋週次レビュー ¥79,800 |
| 特典期限 | 「◯/◯（水）までのお申込みで◯◯」＋受け放題（9月開始予定）先行案内リストのQR |
| フッター | 「本レポートは医療記録ではありません。効果には個人差があります。」 |
