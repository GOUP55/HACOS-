# GOのウェルカムレター（ウェルカムキット同梱用）

> 手書き or 手書き風フォントで印刷。A5サイズ1枚。直筆署名必須。

---

## レター本文

```
                              HACOS LIFESTYLES
                              ────────────────

〇〇さんへ

今日、ここに来てくれてありがとうございます。

正直に言います。
ダイエットは、やり方じゃなくて
「続けられるかどうか」が全てです。

私はずっとそれを考えてきました。

なぜトレーニングしても食事を変えても
ほとんどの人が途中で諦めるのか。

答えは「ひとりだから」でした。

報告できる人がいる。
崩れても戻れる場所がある。
自分のことを本気で考えてくれる人がいる。

それだけで、人は変われる。
本当にそう思っています。

だから私は観音寺でこれをやっています。

あなたの90日間を、私は全力で一緒に歩みます。
数字だけじゃなく、あなたが「変わった」と
感じられる日まで。

一緒に行きましょう。



                              HACOS LIFESTYLES
                              トレーナー  GO
                              
                              ____年____月____日
```

---

## バリエーション（ペルソナ別の差し替え一文）

### LIFE タイプ向け（子育て世代）
```
「忙しい毎日の中で、自分のことを後回しにしてきたんじゃないかと思います。
 今日からは、あなた自身のために使う時間を、一緒に作りましょう。」
```

### EXEC タイプ向け（経営者・管理職）
```
「結果にこだわる人が来てくれました。
 数字で証明します。90日で必ず変わります。」
```

### FREE タイプ向け（フリーランス）
```
「同じタイミングに始まる仲間がいます。
 ひとりじゃないから、続けられます。」
```

## 「4週目に開けてください」封筒の中身

> Week 1のセッション後に渡す。外側に「4週目（約30日後）に開けてください」と書く。

```
〇〇さんへ

今、体重が思うように減らない時期かもしれません。
「自分は向いていないんじゃないか」と
思い始めているかもしれない。

でも、覚えていてください。

あなたが最初に話してくれた言葉。

「_______________________________」

体重の数字は、あなたの全てじゃない。
あなたが変わろうとした理由は、もっと大きい。

明日も、ここに来てください。
それだけでいいです。

                              ── GO
```
