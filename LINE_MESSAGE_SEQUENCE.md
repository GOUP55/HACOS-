# HACOS｜Week1〜12 クライアント向けLINEメッセージシーケンス（全文）

---

## 送信ルール

- **送信者：** GO（HACOS LIFESTYLES公式LINE）
- **文体：** 話し言葉・絵文字あり・温かく・短め
- **送信タイミング：** 各フェーズの指定曜日・時間
- **自動配信と手動の区別：**
  - 🤖 = LINE自動配信（LINE_AUTOMATION.mdと連携）
  - ✋ = GOが手動で送る（個別対応）
