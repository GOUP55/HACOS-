## サプリメント提案の基本哲学

```
「サプリは薬じゃない。食の補完だ。」

HACOSのサプリ提案3原則：

01 必要なものだけ、適切な順番で勧める
   → 全部一気に買わせない。ステップで追加する。

02 食事でとれるなら、まず食事で。
   → 「プロテインより先に鶏むね肉を食べよう」という考え方を伝える。

03 クライアントの予算・生活スタイルに合わせて選ぶ。
   → 高ければいいわけじゃない。継続できるものが正解。
```

## サプリ提案の注意事項

```
⚠️ 禁止事項（景品表示法・薬機法関連）

× 「このサプリを飲めば○kg痩せる」
× 「効果が保証されています」
× 医薬品的な効能・効果の表現

✅ 適切な表現

○ 「タンパク質の補完として活用しています」
○ 「食事だけでは不足しがちな栄養素を補う目的で」
○ 「個人差があります」「食生活・運動と組み合わせて」

⚠️ 医療的な判断が必要な場合

  ・疾患のあるクライアント（糖尿病・腎臓病等）には
    「かかりつけ医に相談してから」と必ず添える
  ・サプリの過剰摂取は逆効果になる場合がある
    （特にビタミンA・D等の脂溶性ビタミン）
```
