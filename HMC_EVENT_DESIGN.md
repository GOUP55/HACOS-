# HMC イベント体験デザインドキュメント
## Healthy Morning Club — Experience Architecture

---

## 1. 体験デザインの哲学

### 基本原則
HMCの90分間は「イベント」ではなく「儀式（リチュアル）」として設計する。
参加者は「消費者」ではなく「旅人」として扱う。
すべての動線・音・香り・空間が、一つのナラティブを体験させるために存在する。

### 体験設計の3層構造
```
Layer 1: 物理的体験（何をするか）
         └── サウナ、食事、音楽、アクティビティ

Layer 2: 感情的体験（何を感じるか）
         └── 解放感、充実、つながり、静けさ

Layer 3: 意味的体験（何を持ち帰るか）
         └── 「今日も自分と向き合えた」という自己効力感
```

## 2. タイムライン設計（90分の儀式）

### マスタースケジュール
```
06:45  [スタッフ] 空間の最終セットアップ・音楽スタート
07:00  ドア開放・参加者受付開始
       └── スマホをロッカーへ（儀式の起点）
07:05  ウェルカムドリンク（スムージーorCBDドリンク）の提供開始
07:10  スタッフによる軽いオープニングコール（2分以内）
       └── 「今日の90分の過ごし方」を案内。講義ではなく詩的に。
07:15  自由時間 START（食事・サウナ・アクティビティへ）
       ├── 食事エリア: ヘルシービュッフェ（立食）
       ├── サウナ/アイスバスエリア: ローテーション制
       └── ムードエリア: シンギングボウル/シーシャ/CBD
08:00  マインドフルネスセッション（おさちさん）15分
       └── 全員集合形式。お茶の所作を用いた「今ここ」への帰還
08:15  クロージング・タイム（自由交流・名刺なし）
08:25  スタッフによるクロージング（1分以内）
       └── 「今日の感覚を、少し持ち帰ってください」
08:30  解散・スマホ返却
```

### タイムライン設計の意図
| 時間帯 | 意図 |
|--------|------|
| 07:00-07:15 | 「入場」の儀式化。スマホを手放すことで非日常が始まる |
| 07:15-08:00 | 自由時間により「選ぶ」体験。でも選択肢は限られている |
| 08:00-08:15 | 全員が同じ時間を共有する唯一の瞬間。一体感の醸成 |
| 08:15-08:30 | 余韻の時間。会話が自然に生まれる |

## 3. 空間設計（ゾーニング）

### 空間の3つのゾーン
```
┌─────────────────────────────────────────┐
│                                         │
│   [A] エントランス/ロッカーゾーン          │
│       スマホ預かり・ウェルカムドリンク       │
│                                         │
│   [B] メインホール（立食・交流ゾーン）      │
│       ヘルシーフード・スムージー             │
│       音楽（少し大きめ） ・CBD提供          │
│                                         │
│   [C] サンクチュアリゾーン               │
│       サウナ・アイスバス                   │
│       シンギングボウル・シーシャ            │
│       マインドフルネス（おさちさん）         │
│                                         │
└─────────────────────────────────────────┘
```

### ゾーン別の空間演出詳細

#### Zone A: エントランス/ロッカーゾーン
```
照明: 暗め（外の朝の光との対比）
香り: ヒノキかパロサント（儀式の香り）
音楽: なし（または小さいアンビエント音）
スタッフ: 1名（静かな笑顔で迎える。言葉は最小限）
設置物: ロッカー、小さな案内板（「スマホを預けると、体験が始まります」）
```

#### Zone B: メインホール（食と交流）
```
照明: 自然光を最大限 + 温かい間接照明
香り: コーヒーとフード（食欲と安心感）
音楽: 少し大きめのハウス/ロー・ファイ/インストジャズ
     （会話はできるが、静寂ではない。クラブ的な非日常感）
レイアウト: 高めのテーブル（立食）、円形配置で会話を促進
食器: 白系の素材感のある器（映えるが主役ではない）
禁止: 椅子、スクリーン、名刺交換を促す仕掛け
```

#### Zone C: サンクチュアリゾーン
```
照明: 暗め・ゆらぎ系（キャンドル的）
香り: ユーカリ/ミント（サウナ側）・白檀（瞑想側）
音楽: バイノーラルビートまたはシンギングボウルの録音（小音量）
温度: サウナ(80〜90℃) → アイスバス(約10℃) の明確な対比
設置物: タオル（HMCロゴ入り）、水、ポスター不要
感覚設計: 「言葉が不要になる空間」を目指す
```

## 4. フードデザイン（体験としての食事）

### コンセプト
「食べることが、瞑想になる朝食」

すべての食材は「視覚的な美しさ」と「身体への優しさ」を同時に満たす。
メニューは固定せず、季節と旬に合わせてシェフが設計する。

### 基本構成
```
① ウェルカムスムージー（CBD入りオプションあり）
   → 到着直後の最初のギフト。色彩的に美しいグリーンかパープル系

② メインフードビュッフェ（3〜4品）
   → 例: グレインボウル / フルーツプレート / 発酵食品 / 良質タンパク

③ コーヒーまたはハーブティー
   → 香りで「朝」を感じさせる

④ マインドフルネスセッション用のお茶（おさちさん選定）
   → 日本茶または薬草茶。所作と共に提供
```

### フードプレゼンテーション原則
- 色彩: 緑・白・黄・紫を基調（視覚的な健康感）
- 器: 木・陶・石のどれか（プラスチック・紙皿禁止）
- 量: 「足りた」ではなく「丁度よかった」を目指す（食べ過ぎさせない）
- 説明: 食材の説明カードを小さく添える（しかし語りすぎない）

## 5. 音楽設計（サウンドスケープ）

### 時間帯別の音楽設計
```
06:45-07:10  [アライバル BGM]
             ジャンル: ロー・ファイヒップホップ / アンビエントハウス
             BPM: 70-90（ゆっくりめ、まだ目が覚めていい）
             音量: 小さめ（50dB程度）

07:10-08:00  [メインタイム BGM]
             ジャンル: ディープハウス / アフロハウス / ミニマルテクノ
             BPM: 100-115（身体が動き出す）
             音量: 少し大きめ（65-70dB、会話は成立するが没入感がある）

08:00-08:15  [マインドフルネス BGM]
             ジャンル: なし、または自然音・シンギングボウルのみ
             音量: 非常に小さいか無音

08:15-08:30  [クロージング BGM]
             ジャンル: 穏やかなジャズ / ボサノバ
             BPM: 80-90（余韻と会話）
             音量: 中程度（会話の邪魔をしない）
```

### プレイリスト参考アーティスト
- Floating Points / Four Tet（アンビエント〜ハウス）
- Nils Frahm（ピアノ・現代クラシック）
- Moby（アンビエント期）
- Jon Hopkins（テクノ×アンビエント）
- LoFi Girl系（アライバル）

## 6. アクティビティ設計

### 定番アクティビティ（各 +1,000円）

#### アクティビティ①: サウナ & アイスバス体験
```
時間: 15〜20分（サウナ8分 → アイスバス2分 × 1〜2セット）
定員: 1セット4〜6名
スタッフ: 1名（安全管理・タイミングの案内）
体験の価値: 「思考の停止」を身体で体験する。最も強烈な感覚の転換
注意事項: 初心者向けの簡単な事前説明を1分で行う
```

#### アクティビティ②: シンギングボウル体験（おさちさん または HMCスタッフ）
```
時間: 15分
定員: 1セット8〜10名
設計: 仰向けで目を閉じ、音の振動を身体で感じる
体験の価値: 「音のサウナ」。視覚を閉じることで聴覚と触覚が増幅
```

#### アクティビティ③: マインドフルネス with お茶（おさちさん）
```
時間: 15分（全員参加のメインセッション）
設計: お茶の香りを嗅ぎ、温度を感じ、一口ずつ飲む
体験の価値: 「今ここ」への帰還。日常の速度を意図的に落とす
```

#### アクティビティ④: シーシャ体験（オプション）
```
時間: 自由（セルフペース）
設計: Zone Cの一角に設置。一人でも、複数でも
体験の価値: 「吐く呼吸」への意識。副交感神経の活性化
注意: フレーバーはニコチンなしのハーブ系を選定
```

## 7. スタッフ設計と役割

### スタッフ体制（20〜30名回）
```
役割              人数    担当
──────────────────────────────────
エントランス担当   1名    受付・スマホ預かり・ウェルカム
フード担当        1〜2名  食事補充・衛生管理・ドリンク提供
サウナ担当        1名    安全管理・ローテーション案内
アクティビティ    1名    シンギングボウル・シーシャ管理
ホスト（MC）      1名    オープニング・クロージング担当
計               5〜6名
```

### スタッフの振る舞い原則
1. **話しかけない** — 参加者が「ひとり」でいる時間を邪魔しない
2. **笑顔で待つ** — 助けを求められたら素早く、求められなければ静かに
3. **服装統一** — ブラック系のシンプルな服（目立たない存在感）
4. **スマホを持たない** — スタッフも「場の一部」として振る舞う

### ホスト（MC）のスクリプト

#### オープニング（2分以内）
```
「おはようございます。HMCへようこそ。

今日の90分は、あなたのものです。
何かをしなくていいし、誰かに合わせなくていい。

ただ、スマホがないという事実だけを
少し楽しんでみてください。

食事はいつでも。サウナは5分後に最初のセットが始まります。
おさちさんのお茶の時間は、8時です。

では、今日も良い朝を。」
```

#### クロージング（1分以内）
```
「今日の朝を、ありがとうございました。

あのサウナの後の静けさ、覚えていますか。
あの感覚だけ、少し持ち帰ってください。

来週またここで、お会いしましょう。

ロッカーのスマホは、お忘れなく。」
```

## 8. 初回（5月18日）特別設計

### ローンチイベントとしての特別仕掛け
```
1. アーリーアダプター特典
   「第1回HMC参加者」の称号（デジタルカード等で後日送付）

2. フォトブース（スタッフが撮影）
   参加者のスマホは禁止でも、HMCスタッフが特定シーンで撮影
   → 参加後、メールで写真を送付（SNS投稿の種を作る）

3. 特別ゲスト招待
   地元のウェルネス系インフルエンサーや経営者を数名招待
   → 口コミ拡散の起爆剤に

4. アンケートの代わりに「一言メモ」
   退出時に小さな紙に「今日感じたこと」を一言書いてもらう
   → 壁に貼る/後日SNSで紹介（参加者の声を可視化）
```

### 初回の目標KPI
```
参加者数:    20名（最低）〜 30名（理想）
リピート意向: 70%以上（「また来たい」と感じた人）
口コミ発生:  参加者の30%以上が誰かに話す
SNS投稿:    5件以上（スタッフ写真の提供で促進）
```

## 9. リスク対策と品質管理

### 想定リスクと対処
| リスク | 対処 |
|--------|------|
| 参加者がスマホを拒否 | 事前に予約時に「スマホなし体験」を明記。当日は強制ではなく、「お預かりオプション」として案内 |
| サウナの混雑 | ローテーション制を厳密に管理。待ち時間には食事やシンギングボウルへ誘導 |
| 初対面の緊張感 | ウェルカムドリンクを手渡しで提供。「物」を介した自然な接触を設計 |
| 雨天・悪天候 | 屋外要素がある場合は代替プランを事前準備 |
| 参加者の健康トラブル | サウナ利用前の簡単な体調確認。緊急連絡先とAEDの場所を全スタッフが把握 |

### 体験品質チェックリスト（毎回）
```
□ 音楽はタイムライン通りに設定されているか
□ フードは開場10分前に美しく並んでいるか
□ サウナの温度は指定温度に達しているか
□ ウェルカムドリンクは十分な数が用意されているか
□ ロッカーは参加者全員分確保できているか
□ スタッフ全員が役割を把握しているか
□ 緊急時の連絡体制は整っているか
```

## 10. 体験後のエコシステム設計

### 参加者が体験後に行動する仕掛け
```
当日:    HMCスタッフ撮影写真をメールで送付
         → 「今日の朝」を可視化。SNS投稿の動機づけ

翌日:    フォローアップメール送信
         → 「今日の感覚を、少し持ち帰ってください」

1週間後: 次回開催のリマインド
         → 前回の写真1枚と共に「また来週」の告知

1ヶ月後: 月間参加者への感謝メール
         → 「HMCレギュラー」として特別扱いのシグナル
```

### コミュニティ・ループの設計
```
体験 → 感動 → 口コミ → 新規参加 → 体験

このループが回り始めた時、
広告費ゼロでもHMCは成長し続ける。
```
