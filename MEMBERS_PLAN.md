# 月額制プラン「HACOSメンバーズ」設計書（案）

> 収益設計TOP5・第5位。対象：HACOS 3ヶ月プレミアムメソッド 第1期生（定員3名・10月卒業想定）の卒業後継続プラン。
> 金額・仮定は全て（仮）。実施前にオーナー確認必須。

## ◆ 統合決定（2026-07-07 オーナー決定）

`BUSINESS_RULES.md` にあった別構想「通い放題 ¥20,000〜/月（準備中）」は、**本メンバーズプランに統合**する。

- 「HACOSメンバーズ」を月額制プランの正式名称として一本化し、**卒業生専用ではなく一般向けにも開く**
- 一般向け＝通常価格でメンバーズに加入（旧「通い放題」の受け皿）
- プレミアム卒業生＝LP約束の20%OFF優待（卒業生スタート価格・初回3ヶ月）で同じプランに加入
- 「通い放題」という別商品は今後作らない（名称としてもLINE・LPで使わず「メンバーズ」に統一）
- 一般向けの含むもの・価格差（卒業生との公平性）の詳細は下記オーナー決定事項に追加

この統合により、10月の卒業生案内を待たず**一般向け先行案内（LINE）から開始できる**。

## ◆ 前提：premium.html で既に約束済みの内容（矛盾させない）

`premium.html` の特典セクション（掲載協力で卒業後もずっとお得に）に、以下がすでに明記・公開されている。

- 卒業後もパーソナルを **20%OFF** で継続（**3ヶ月間**）
- HMC は **会員価格を据え置き** でずっと参加OK
- 第1期 卒業生**コミュニティに継続参加**
- 上記はすべて **「ビフォーアフター・データの掲載協力」への任意の返礼特典**（掲載しなくても受講内容・料金に影響なし）

→ **本設計書はこの約束を前提として組み込む。** 特に「20%OFF・3ヶ月間・パーソナル部分限定・掲載協力任意」という条件は変更せず、メンバーズプランの価格にそのまま反映する（下記「卒業生スタート価格」参照）。また「卒業生コミュニティ」はメンバーズ加入の有無にかかわらず、掲載協力した卒業生には無償で提供済みの特典として扱う（メンバーズの有料特典としては数えない）。

## ◆ プラン設計案（2プラン）

前提の目安料金（`premium.html` 比較表より）：パーソナル ¥6,250〜/回、HMC ¥2,000/回。

### ライト（仮：月¥20,000）

| 項目 | 内容 |
|------|------|
| パーソナル | 月2回（マンツーマン） |
| HMC | 毎週参加（会員価格据え置き） |
| LINEサポート | 週1回程度のゆるい相談チャット |
| 相当額の目安 | パーソナル ¥12,500 相当＋HMC ¥8,000 相当（週4回換算）＝計¥20,500 相当 |
| 会員価格（仮） | **¥20,000/月** |

### スタンダード（仮：月¥29,800）

| 項目 | 内容 |
|------|------|
| パーソナル | 月4回（マンツーマン） |
| HMC | 毎週参加（会員価格据え置き） |
| LINEサポート | 毎週の個別相談チャット |
| その他 | 隔月の簡易体組成チェック（要オーナー判断・実施可否） |
| 相当額の目安 | パーソナル ¥25,000 相当＋HMC ¥8,000 相当＝計¥33,000 相当 |
| 会員価格（仮） | **¥29,800/月** |

### 卒業生スタート価格（掲載協力・卒業後3ヶ月限定・仮）

premium.html の「パーソナル20%OFF・3ヶ月間」を、メンバーズのパーソナル相当部分にのみ適用した金額。HMC部分は「会員価格据え置き」ですでに割引済みの前提のため、追加の割増引きはしない。

| プラン | 卒業後1〜3ヶ月目（仮） | 4ヶ月目以降（通常価格） |
|--------|----------------------|------------------------|
| ライト | ¥17,500（パーソナル相当¥12,500×80%＋HMC¥8,000） | ¥20,000 |
| スタンダード | ¥26,000（パーソナル相当¥25,000×80%＋HMC¥8,000） | ¥29,800 |

※ 掲載協力していない卒業生は、最初から通常価格（¥20,000／¥29,800）でのご案内となる。単発パーソナル継続のみを希望する掲載協力者には、別途「単発20%OFF＝¥5,000/回（3ヶ月間）」の従来案内も並行して用意し、卒業生が「まとめてメンバーズ」か「単発継続」かを選べるようにする（決定事項#5）。

## ◆ 収益インパクト試算（2〜3ケース・全て仮定）

**前提の断り書き：** 第1期生はまだ卒業していない（本日2026-07-06時点、卒業想定10月）。継続率・実際の選択プランは未知数。以下は「もし」の試算であり、実績・見込みではない。

### ケースA：3名とも「ライト」を12ヶ月継続した場合（仮）

- 1名あたり：卒業生スタート価格3ヶ月（¥17,500×3=¥52,500）＋通常9ヶ月（¥20,000×9=¥180,000）＝**¥232,500**
- 3名合計：**¥697,500**（仮・12ヶ月間の想定売上）

### ケースB：3名とも「スタンダード」を12ヶ月継続した場合（仮）

- 1名あたり：卒業生スタート価格3ヶ月（¥26,000×3=¥78,000）＋通常9ヶ月（¥29,800×9=¥268,200）＝**¥346,200**
- 3名合計：**¥1,038,600**（仮）

### ケースC：現実的な混在シナリオ（仮）

- 1名：スタンダードを12ヶ月継続 → ¥346,200
- 1名：ライトを6ヶ月継続後に退会 → 卒業生スタート価格3ヶ月（¥52,500）＋通常3ヶ月（¥60,000）＝¥112,500
- 1名：メンバーズには加入せず継続なし → ¥0
- 合計：**¥458,700**（仮）

## ◆ 未確定・要フォローの論点

- 「単発20%OFF継続」と「メンバーズの卒業生スタート価格」を両方用意すると案内が複雑になる可能性がある。第1期卒業時に実際どちらが選ばれるかを見て、第2期以降はシンプルな方に寄せる判断をしてよい。
- 体組成測定・簡易チェックの実施可否（スタンダード案に含めるか）は運用負荷次第でオーナー判断。
- 初期メンバー特典（決定事項#3）は、第1期生3名という少人数だからこそ効く可能性がある一方、原価インパクトも小さいため検討コストは低い。
