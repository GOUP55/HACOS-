# HACOS Wellness Journey — 安全・キャンセル・中止ルール（規約ドラフト）v1.0

> 状態: **ドラフト（GO承認前）**。承認後にLP・申込フォーム・同意書へ同じ文言を反映し、表記を統一する。
> 参加費のキャンセル区分は「運動・サポート部分」と「食事部分」で扱いが異なる点が肝。

## 1. 開催・中止

| 項目 | ルール（案） |
|---|---|
| 最少催行人数 | 3名（**全プログラム共通**・07-14決定）。申込締切（第1期: **2026-07-31**）時点で判定 |
| 支払方法 | **現金またはSTORES決済**（07-14決定） |
| 席の確定 | **申込＝仮予約。申込後3日以内の入金で席確定**（07-14決定）。催行判定（7/31）は入金済み人数で行う |
| 3名未達の場合 | 開催週を順延して再調整。順延できない場合は**全額返金** |
| 雨天 | 中止にせず屋内プログラム（ストレッチ・コンディショニング）に変更 ※会場は要決定 |
| 警報・災害 | 気象警報（暴風・大雨等）発令時は当日の朝プログラムを中止。LINEで当日5:30までに連絡。振替または内容で代替 |
| 主催者都合の休講 | 振替を行う。振替できない場合は該当分を返金 |

## 2. キャンセル・返金（参加者都合）

| タイミング | 運動・サポート部分 | 食事部分 |
|---|---|---|
| 申込締切（7/31）まで | 全額返金 | 全額返金 |
| 締切後〜食事製造開始前 | 要決定（案: 50%） | 全額返金 |
| **食事製造開始後**（開催5日前〜） | 要決定（案: 50%） | **原則返金不可**（受注生産のため） |
| 開始後の途中離脱 | 返金なし（案） | 返金不可。引き渡し済みの食品はお渡し済み扱い |

- 体調不良・怪我による離脱は個別協議（無理な参加を防ぐため、ペナルティ的な扱いにしない）

## 3. 欠席・食品の受け取り

- 欠席日の弁当・間食は**当日中の受け取り**が基本。受け取れない場合、安全上の理由から翌日への持ち越しは保証しない
- 冷凍・冷蔵商品の受け取り後の保存は参加者の責任（保存方法はDay0に説明・紙でも渡す）
- 体調不良時は無理に食べず、GOへ連絡する

## 4. 健康・安全

- 申込時に健康チェック（PAR-Q型）を実施。「はい」がある場合は個別相談とし、必要に応じてかかりつけ医への相談をお願いする（**運営は医学的判断・診断をしない**）
- 運動中の体調変化は即申告。無理をさせない運営を最優先する
- 傷害保険: スポーツ安全保険等への加入を検討（**要決定**）
- 緊急連絡先を申込時に必ず取得
- 早朝の屋外活動のため、集合場所の安全（照明・車通り）を事前確認

## 5. 個人情報・写真

- 健康情報（体重・測定値・既往歴）は運営メンバーのみ閲覧。参加者間・法人へは本人同意なく共有しない
- 法人プランでは匿名化・集計済みデータのみ企業へ提出
- 写真・体験談の利用（SNS・LP掲載）は**別途の同意**を取り、同意範囲（グループ内のみ／公開OK）を選べるようにする
- データの保存期間と削除方法を定める（**要決定**: 案・終了後1年で削除、継続会員は在籍中保持）

## 6. 表現ルール（全資料共通・確定済み方針）

- 医療行為と誤認される表現を使わない／治療・診断・治癒を約束しない
- 「ウェルネスレポート（旧称カルテ）」には「医療記録ではありません」と明記
- 効果には個人差があることを明記
- 持病・医療的管理が必要な場合は医師へ相談を促す

# 保健所 確認事項リスト（募集開始前・必須）

> 窓口: 観音寺市を管轄する保健所（香川県 西讃保健福祉事務所）。GOまたは食事担当が電話→必要なら訪問。
> **申込締切7/31から逆算し、7/18頃までの確認を強く推奨**（NGだった場合に食事構成を組み替える時間を確保するため)

## 聞くことリスト

1. **営業許可の範囲**: 現在の飲食店営業許可で「冷凍おかずセット（受注生産・手渡し）」の販売が可能か。冷凍食品製造業等の追加許可が必要か
2. **製造場所の条件**: 現在の厨房で冷凍販売用の調理・急速冷凍・保管をしてよいか
3. **食品表示**: 対面手渡し・受注生産の場合の表示義務（原材料・アレルゲン・保存方法・消費期限・製造者）はどこまで必要か
4. **アレルギー表示**: 特定原材料の表示方法
5. **保存期限の設定**: 冷凍おかずの消費期限は自社で設定してよいか。根拠(検査)は必要か
6. **加熱方法表示**: 「要加熱」表示の要否
7. **保冷・輸送**: 保冷バッグでの持ち帰り（車移動前提）に関する注意点
8. **弁当・間食・スムージー**: 既存の弁当営業と同じ扱いでよいか（提供場所が屋外集合場所になる点）
