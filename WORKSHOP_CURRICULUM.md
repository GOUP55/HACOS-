# HACOS 発酵食ワークショップ カリキュラム完全版
## 90日コホート Month 1〜3 詳細手順書

> 各コホートで実施する3回の発酵食ワークショップ。
> GOが実施するか、提携農家・料理家を招いて実施する。
> 所要時間：各回 90分（実習60分 + 試食・質疑30分）

## Month 1 ワークショップ：「塩糀を仕込む」

> テーマ：発酵の入口。毎日の食事を簡単に整える魔法の調味料。

### 目的

```
・発酵食品を「難しいもの」から「日常のもの」へ変える
・塩分を減らしながらうまみを増やす感覚を体験する
・「自分で作れる」という自己効力感をつくる
```

### 準備物リスト（5名分）

```
【材料】
  乾燥米糀：500g（スーパーやネットで購入可）
  塩：150g（天然塩・粗塩を使うとうまみが増す）
  ぬるま湯：150〜200ml（50〜60℃が理想）

【道具】
  清潔な保存瓶（300ml程度）×5個
  大きめのボウル ×1
  計量スプーン・計量カップ
  木べら or スプーン
  ゴム手袋（任意）
  温度計

【持ち帰り用】
  説明カード（今日仕込んだ分の熟成期間・管理方法を記載）
  レシピカード（塩糀を使った3品：鶏肉の塩糀漬け、塩糀のドレッシング、塩糀卵）
```

### タイムライン（90分）

```
00:00〜10:00 ｜ イントロダクション
  ・発酵と腸内環境の関係（なぜ発酵食品が脂肪燃焼に効くのか）
  ・米糀とは何か（見て、触って、においを嗅ぐ）
  ・「引く食より足す食」のHACS食哲学との接続

10:00〜45:00 ｜ 仕込み実習
  Step1：糀をほぐす
    ・ボウルに乾燥米糀を入れる
    ・手でほぐして、粒粒をバラバラにする（5分）
    ・「糀の甘い香り」に注目させる

  Step2：塩を混ぜる
    ・糀に塩150gを加える
    ・全体がまんべんなく混ざるまでよく混ぜる（3分）
    ・混ぜながら「引く食の塩分制限」との違いを話す

  Step3：ぬるま湯を加える
    ・50〜60℃のぬるま湯を少しずつ加える
    ・べたつかず、全体がしっとりする程度まで（水分量は好み）
    ・温度が高すぎると糀菌が死ぬ→温度計で確認

  Step4：瓶に詰める
    ・清潔な保存瓶に詰める
    ・空気が入らないように軽く押し込む
    ・フタは「ゆるく」閉める（ガスが出るため）

  Step5：ラベルを貼る
    ・今日の日付と「解禁日（10〜14日後）」を書く

45:00〜60:00 ｜ 試食タイム
  ・去月GOが仕込んでおいた完成塩糀を試食
  ・塩糀ドレッシングのサラダ
  ・塩糀漬け鶏肉のグリル（試食用）

60:00〜80:00 ｜ 使い方レクチャー
  ・鶏肉の塩糀漬け（一番使いやすい基本レシピ）
  ・塩糀ドレッシングの作り方
  ・「いつものレシピの塩を塩糀に変えるだけ」の実践方法

80:00〜90:00 ｜ 質疑・片付け
  ・来月のワークショップ予告
  ・今日の糀を持ち帰り、1週間後にLINEで報告してもらうよう依頼
```

### 熟成期間中の管理方法（渡すカード）

```
塩糀の育て方

室温保存で10〜14日で完成します。

毎日1回、清潔なスプーンでかき混ぜてください。
（空気を入れる → 発酵が進む）

夏：10日で完成することも
冬：14〜20日かかることも

完成のサイン：
  ・糀の粒が柔らかくなる
  ・甘い発酵臭がする
  ・全体がとろっとしてくる

完成したら冷蔵庫へ。3〜6ヶ月保存可能。
```

### 使い方レシピカード（持ち帰り）

```
【塩糀でできる3品】

① 鶏むね肉の塩糀漬け
  材料：鶏むね肉200g・塩糀大さじ1
  方法：肉に塩糀を揉み込み、一晩置いてグリルするだけ
  効果：肉が柔らかくなる・うまみが増す・タンパク質吸収UP

② 塩糀ドレッシング
  材料：塩糀大さじ1・オリーブ油大さじ2・酢大さじ1・にんにく少々
  方法：全て混ぜるだけ
  効果：腸活 + 抗酸化 + うまみで減塩に

③ 塩糀漬け卵（常備菜）
  材料：ゆでたまご3個・塩糀大さじ2
  方法：ビニール袋に入れて1〜2日置くだけ
  効果：タンパク質+発酵の最強常備菜
```

## Month 2 ワークショップ：「味噌汁を再設計する」

> テーマ：毎日の一杯を「最高の腸活ドリンク」に変える。

### 目的

```
・毎朝の味噌汁が習慣になれば、腸内環境が変わることを理解する
・自分専用の「HACOS版パーソナル味噌汁」をデザインする
・発酵食の継続ハードルを「毎日の朝ごはん」に埋め込む
```

### 準備物リスト（5名分）

```
【材料】
  味噌各種（白味噌・赤味噌・麦味噌）の食べ比べセット
  だし素材：煮干し・昆布・鰹節
  具材セット：豆腐・わかめ・油揚げ・ねぎ・きのこ類・根菜
  トッピング：えごまオイル・すりごま・海苔

【道具】
  鍋 ×3〜5台（ガスコンロまたはIH）
  だし取り用鍋・ザル
  器 × 5個
  計量スプーン

【持ち帰り用】
  「自分だけの味噌汁レシピカード」（当日記入→持ち帰り）
  おすすめの地元味噌・だし素材リスト（購入先付き）
```

### タイムライン（90分）

```
00:00〜15:00 ｜ イントロダクション
  ・味噌汁とダイエットの科学（腸→脳→食欲制御のメカニズム）
  ・「朝の味噌汁」が睡眠の質を上げる理由（トリプトファン→セロトニン→メラトニン）
  ・讃岐と発酵食文化のつながり（香川の麦味噌など）

15:00〜30:00 ｜ だし取り体験
  ・水出し昆布だしを触る・においを嗅ぐ
  ・煮干しだしと鰹節だしの飲み比べ
  ・「うまみ = グルタミン酸」の感覚を味で理解する

30:00〜60:00 ｜ 自分だけの味噌汁をつくる実習
  Step1：だしを選ぶ（昆布 / 煮干し / 鰹節 / ミックス）
  Step2：具材を選ぶ（腸活視点でのセレクト）
    → タンパク質：豆腐・油揚げ
    → 食物繊維：きのこ・わかめ・ごぼう
    → ビタミン・ミネラル：ねぎ・根菜
  Step3：味噌を選ぶ（白・赤・麦の食べ比べ後に決める）
  Step4：仕上げのひと工夫
    → えごまオイルを数滴（オメガ3添加）
    → すりごま（抗酸化）
    → 卵を落とす（タンパク質＋）

60:00〜75:00 ｜ 試食 + レシピカード記入
  ・それぞれが作った味噌汁をシェア
  ・「自分だけの味噌汁レシピ」をカードに書いて持ち帰る

75:00〜90:00 ｜ 「毎朝続けるコツ」レクチャー
  ・前日夜に具材を切っておく（2分で完成）
  ・保温ポットに入れて持ち歩く方法
  ・週末にだしパックを作っておく
  ・来月の酵素シロップへの橋渡し
```

## Month 3 ワークショップ：「酵素シロップを作る」

> テーマ：果物と砂糖と時間が生み出す、発酵の奇跡。

### 目的

```
・90日の集大成として「自分で作れる発酵食品」を増やす
・甘いものへの欲求を「罪悪感なく満たす方法」を知る
・卒業後も家で続けられる発酵習慣を一つ持ち帰る
```

### 準備物リスト（5名分）

```
【材料（1人分）】
  旬の果物：300g（いちご・キウイ・りんご・ゆずなど季節のもの）
  砂糖：300g（きび砂糖・甜菜糖 → 白砂糖より精製度が低いもの）
  ※ 果物と砂糖は1:1の重量

【道具】
  清潔な瓶（500ml以上）×5個
  包丁・まな板
  計量スケール
  ゴム手袋（任意）
  木べら

【持ち帰り用】
  仕込み済みシロップ（持ち帰って熟成させる）
  管理カード（熟成期間・毎日のかき混ぜ方）
  活用レシピカード（ドリンク・ドレッシング・ヨーグルト添え）
```

### タイムライン（90分）

```
00:00〜15:00 ｜ イントロダクション
  ・「酵素」とは何か（食材に含まれる自然の発酵力）
  ・砂糖が「防腐剤」から「発酵の栄養」に変わる仕組み
  ・90日の振り返り：Month1（糀）→ Month2（味噌汁）→ Month3（シロップ）

15:00〜50:00 ｜ 仕込み実習
  Step1：果物を洗う・乾かす（水分がカビの原因になる→丁寧に）
  Step2：果物を薄切りにする（表面積を大きくすることで発酵が進みやすくなる）
  Step3：瓶に果物と砂糖を交互に重ねる
    → 一番下：砂糖
    → 果物を並べる
    → 砂糖をかける
    → 果物を並べる
    → 一番上：砂糖で蓋をする
  Step4：フタをゆるく閉める
  Step5：日付と解禁日（10日後）をラベルに書く

50:00〜70:00 ｜ 試食 + ドリンク体験
  ・GOが事前に仕込んでおいた完成シロップを使ったドリンク
    → 炭酸水で割ったシロップドリンク
    → ヨーグルトにかけてパフェ風
    → ドレッシングに混ぜてサラダ
  ・「甘いもの欲求のリダイレクト」を体感する

70:00〜85:00 ｜ 卒業後の発酵生活 設計タイム
  ・各自「卒業後も続ける発酵習慣ひとつ」を決める
  ・プランブックの最終ページ「次の90日の目標」に記入
  ・グループで発表・シェア

85:00〜90:00 ｜ クロージング
  ・GOから全員へのメッセージ（手書きカード or 口頭）
  ・Googleレビュー依頼カード配布
  ・集合写真（希望者）
  ・次コホートの案内
```

### 酵素シロップ 管理カード（持ち帰り）

```
酵素シロップの育て方

【熟成場所】
  常温・直射日光の当たらない場所

【毎日のケア】
  1日1回、清潔な手でかき混ぜる（または瓶をゆする）
  これで空気が入り発酵が進む

【完成のサイン（10〜14日後）】
  ・砂糖がほぼ溶けている
  ・発酵したやや酸っぱい甘い香り
  ・小さな泡が出ている

【完成したら】
  果物を取り出して、シロップだけ保存
  冷蔵庫で 1〜2ヶ月保存可能

【飲み方（1日 大さじ1〜2が目安）】
  炭酸水で割る（1:4）→ 最高の腸活ドリンク
  ヨーグルトにかける
  ドレッシングに混ぜる
```
