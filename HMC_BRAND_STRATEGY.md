# HMC ブランド戦略ドキュメント
## Healthy Morning Club — Brand Strategy

---

## 1. ブランド・アイデンティティ

### ブランド名とポジショニング
- **正式名称:** Healthy Morning Club（HMC）
- **読み方:** ヘルシー・モーニング・クラブ
- **ポジション:** 四国初の「五感リカバリー型・プレミアム朝会」

### ブランド・エッセンス（1行）
> **「あなたの魂に、月曜日の朝を味方にする」**

### ブランド・パーパス（存在意義）
情報とアルゴリズムに支配された現代人に、自分自身の感覚を取り戻す「オフライン・サンクチュアリ」を提供する。

### ブランド・パーソナリティ
| 属性 | 描写 |
|------|------|
| 温度感 | 温かく、でも洗練されている |
| 言葉のトーン | 静かな自信。押しつけず、誘う |
| 見た目の印象 | アーバン禅。モノトーン × 緑と木の質感 |
| 象徴的な人物 | 週末はサウナ、平日は経営判断を下す40代 |

## 2. ターゲット・インサイト

### プライマリーターゲット
**「意思決定疲弊層」の大人**
- 年齢: 30代後半〜50代
- 属性: 経営者・管理職・フリーランス・クリエイター
- 居住: 高松市・丸亀市・坂出市など広域
- 心理: SNSを開くたびに疲れる。朝から情報に追われている。体は動かしたいが、ジムは続かない。

### セカンダリーターゲット
**「自己投資意欲の高いウェルネス感度層」**
- 年齢: 25〜35歳
- 属性: ヨガ・サウナ・マインドフルネスなどに既投資
- 動機: 新しい「場」と「仲間」を探している

### ターゲットの深層欲求（Jobs to Be Done）
```
機能的ニーズ:  健康的な食事を取りながら体を動かしたい
感情的ニーズ:  「今日の自分は良かった」という充足感が欲しい
社会的ニーズ:  同じ価値観を持つ人と、リアルにつながりたい
変革的ニーズ:  月曜日の朝を「憂鬱」から「儀式」に変えたい
```

## 3. ブランド・ナラティブ（物語の構造）

### マスター・ストーリー
```
【現状の世界】
平日の朝7時。スマホのアラームで目覚め、
寝ぼけ眼でSNSのタイムラインを流し見る。
ニュース、広告、他人の成功、政治の不安——
まだ布団の中なのに、すでに意思決定を迫られている。

【問題の特定】
これは疲労ではなく、「感覚の摩耗」だ。
私たちは情報を消費しすぎて、自分の感覚を信じることを忘れた。

【解決の提示】
月曜日の朝だけ、スマホを置いていく。
そこには香ばしいコーヒー、温かい食事、
汗と笑顔と、言葉を超えた人とのつながりがある。

【新しい世界】
HMCの朝を経験した人は言う——
「これが本当の意味での、一週間のリセットだった」
```

### ブランドの3つの「させない約束」
1. スマホを使わせない（代わりに、五感を使う）
2. 選ばせない（メニューはシェフが決める、アジェンダはHMCが設計する）
3. 急がせない（90分間、時間を忘れる空間設計）

## 4. 体験の4柱とブランド価値の紐付け

### 柱①：ご褒美ヘルシー（Feel Good × Does Good）
**ブランドメッセージ:** 「美しいものが、体に良い」

- 立食スタイル = 自由と流動性。座らないことで会話が生まれる
- ヘルシーフード + スムージー = 「我慢」ではなく「恵み」として提供
- 視覚的な美しさ = 料理は「映える」のではなく「感動させる」もの
- **価格正当化:** 食事込み5,000円は「健康への投資」であり「時間の贅沢」

### 柱②：感覚のコントラスト（極端な対比効果）
**ブランドメッセージ:** 「身体のスイッチを、強制的に切り替える」

- サウナ → アイスバス = 日常の思考回路をリセットするプロトコル
- 温冷交代浴の概念をブランド体験全体に応用
- 「熱」の後の「冷却」= 緊張の後の弛緩 = モーニングルーティンの極致
- **体験の希少性:** 早朝7時にサウナ＋アイスバスができる空間は四国に存在しない

### 柱③：ムード・チューニング（気分の調律）
**ブランドメッセージ:** 「静と動、両方がある。だから整う」

- CBD = 抗炎症・鎮静。科学的ウェルネスを体験として届ける
- シンギングボウル = 音の振動で自律神経を整える「音のサウナ」
- シーシャ = 呼吸を意識させる儀式。「吐く」ことで脳がオフになる
- マインドフルネス（おさちさん）= お茶の所作で「今ここ」に帰還する
- **大人のクラブ空間:** 音楽は「少し大きめ」で非日常を演出。でも会話は聞こえる音量設計

### 柱④：選択からの解放（デジタルデトックスの儀式化）
**ブランドメッセージ:** 「スマホを置いてくることが、最高の贅沢になる朝」

- スマホ禁止 = ルールではなくプレミアム体験の「入場券」
- AIと情報社会からの90分間の離脱 = 意思決定疲弊からのリカバリー
- **ポジショニング:** 五感で感じる体験は、画面上では代替できない唯一の価値

## 5. 価格戦略とバリュー・プロポジション

### 価格設計の哲学
```
ドロップイン: 5,000円（食事込み）
アクティビティ: 各 +1,000円
アルコール持込: +1,000円
```

**価格正当化のフレーミング:**
- スターバックスのグランデラテ ≒ 600円 × 8 = 4,800円（1週間分）
- その1週間のコーヒー代で、「完全に整う月曜の朝」が買える
- 高級スパ・エステの相場は1回 15,000〜30,000円。HMCはその1/3以下
- ビジネスランチ1回分の投資で、「週の最初」を最高の状態にリセット

### ターゲット別価値訴求
| ターゲット | 訴求ポイント |
|-----------|------------|
| 経営者・管理職 | 意思決定の質を上げるための「頭のクリーニング」 |
| フリーランス | 孤独な仕事の合間に「本物の人間とつながる場」 |
| ウェルネス層 | サウナ＋マインドフルネス＋良質な食が一度に揃う |
| 高松市からの参加者 | 徳島・川寺エリアへの「プレミアムな遠征価値」 |

## 7. コミュニティ設計と体験の再現性

### 参加者ジャーニーマップ
```
[告知]     SNS/口コミ → 「こんな朝があるのか」と気づく
[予約]     予約フォーム → 限定感と期待感の醸成
[前夜]     リマインドメール → 「スマホを置いてきてください」という予告
[当日朝]   到着 → スマホをロッカーへ（儀式の開始）
[体験]     90分間の感覚の旅
[解放]     スマホ返却 → 「また来週も来たい」という余韻
[拡散]     体験後の感想共有（口コミの自然発生）
```

### コミュニティ化の仕掛け
- **名刺禁止・肩書禁止:** 職業や役職ではなく「人」として出会う
- **シーズナルなメニュー変更:** 季節ごとに来る理由を作る
- **常連優待:** 月4回参加者には「HMCメンバー」称号と特典
- **紹介制度:** 「あなたが連れてきた人」という物語が生まれる設計

## 8. 競合分析とポジショニング・マップ

### 競合カテゴリー別分析
| カテゴリー | 代表例 | HMCとの差分 |
|-----------|--------|------------|
| 朝活コミュニティ | 朝渋・モーニングピッチ | ビジネス志向。HMCは感覚・身体が主軸 |
| ウェルネス施設 | スパ・サウナ施設 | 個人体験。HMCはソーシャルな共体験 |
| 高級ブランチ | ホテルビュッフェ | 食が主役。HMCは体験全体のデザイン |
| マインドフルネス | ヨガスタジオ | 静的。HMCは「動と静」の対比を設計 |

### 唯一無二のポジション
```
「身体で感じる朝の社交場」

高い体験価値
       ↑
個人体験 ←————————————————→ 社交体験
       |              ★HMC
       ↓
低い体験価値
```

HMCが占めるのは「高い体験価値 × 社交体験」の象限。この領域に四国のプレイヤーは存在しない。

## 9. KPI設計と成長フェーズ

### フェーズ1（ローンチ〜3ヶ月）: 5月〜7月
- 目標: 毎回20名以上の参加達成
- KPI: リピート率 40%以上、口コミ獲得数 月10件以上
- 施策: 体験者レポートの蓄積、SNSの世界観構築

### フェーズ2（認知拡大 4〜6ヶ月）: 8月〜10月
- 目標: 高松からの参加者 30%以上
- KPI: 予約キャンセル待ち発生、メディア掲載1件以上
- 施策: インフルエンサー招待回の開催、プレス向け体験会

### フェーズ3（ブランド確立 7〜12ヶ月）
- 目標: 定員50名回の定常開催
- KPI: 法人スポンサー獲得、他都市展開の打診受領
- 施策: HMC年間会員権の販売、ポップアップ開催（高松・大阪）

## 10. HMCが目指す「究極の状態」

> **「HMCに来た月曜日だけ、人生で一番良い月曜日になる」**

参加者が体験後に感じてほしい感情:
1. 「こんな朝の使い方があったのか」という発見
2. 「また来週も来たい」という能動的な意欲
3. 「あの人にも教えてあげたい」という共有衝動

これが生まれた時、HMCはブランドではなく**コミュニティの象徴**になる。
