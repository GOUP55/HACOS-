# HACOS スペースシェア 条件設計

> フリーランサーが「ここをホームにしたい」と思える仕組みをつくる

---

## 1. HACSがスペースシェアで提供するもの（価値の整理）

パートナーが受け取るのは「場所」だけではない。

| 提供価値 | 内容 |
|---------|------|
| スペース・設備 | トレーニング機器・環境一式 |
| ブランド | HACOS の世界観・コンセプト |
| コミュニティ | 他のパートナーとの繋がり・相互紹介 |
| 集客サポート | HACSのSNS・既存会員への紹介 |
| イベント参加 | 日曜ドロップイン等のコラボ機会 |
| 信頼感 | 「ちゃんとした場所」でやっているという安心感 |

---

## 2. パートナー対象ジャンル

HACSのコンセプト「体を整え、体験する」に合うサービス。

**優先ジャンル**
- ヨガ・ピラティスインストラクター
- パーソナルトレーナー（専門特化型）
- 整体・ボディケア施術者
- コンディショニング・ストレッチ専門

**拡張ジャンル（将来）**
- サウナ・リトリート企画者
- 栄養士・食事コーチング
- メンタル・マインドフルネス系

---

## 3. 採用プラン：プランA（売上シェア型）【確定】

```
HACSがもらう = パートナーの売上 × 20%
```

| 項目 | 内容 |
|------|------|
| シェア率 | **売上の20%** をHACOSへ |
| 支払いタイミング | 月末締め・翌月払い（月次精算）|
| 対象 | パートナーがHACS内で行ったセッション全て |

**収益シミュレーション例：**

| パートナーの月売上 | HACSへの入金 |
|-----------------|------------|
| 50,000円（月10セッション×5,000円）| 10,000円 |
| 100,000円（月10セッション×10,000円）| 20,000円 |
| 150,000円（月15セッション×10,000円）| 30,000円 |

---

### 将来の移行プラン（参考）

| フェーズ | 目安 | プラン |
|---------|------|--------|
| 初期（今〜3ヶ月）| 仲間づくり期 | プランA：売上シェア20% |
| 中期（3〜6ヶ月）| 関係安定後 | プランC：最低1.5万円＋シェア15% |
| 長期（6ヶ月〜）| コミュニティ自走後 | プランB：月額固定（個別設定）|

## 4. シェア可能時間帯【確定】

```
日曜日  ：終日シェア可能
月曜日  ：終日シェア可能
火〜金曜：7:00〜9:30のみ（早朝枠）
土曜日  ：17:00以降（夕方〜夜枠）
```

### 曜日別タイムテーブル

| 曜日 | シェア可能時間 | 特徴・向いてるサービス |
|------|-------------|-------------------|
| 日曜 | 終日 | ドロップイン・グループ系・体験会 |
| 月曜 | 終日 | まとまった時間が取れる・施術系に最適 |
| 火〜金 | 7:00〜9:30 | 早朝ヨガ・朝活トレーニング向け |
| 土曜 | 17:00〜 | 仕事帰りのセッション・夜クラス向け |

### パートナー別おすすめ時間帯

| サービスジャンル | おすすめ曜日・時間 |
|--------------|----------------|
| ヨガ・ピラティス | 火〜金 早朝（7:00〜9:30）/ 日曜 |
| 整体・ボディケア | 月曜 終日 / 土曜 17:00〜 |
| パーソナルトレーナー | 月曜 終日 / 火〜金 早朝 |
| グループ体験・ワークショップ | 日曜 終日 |

## 5. 基本条件・ルール

### パートナーに求めること

**必須条件**
- [ ] HACSのコンセプト・雰囲気に共感できること
- [ ] 自身のSNS・発信があること（相互発信が基本）
- [ ] 利用時のスペース・設備の管理責任を持てること
- [ ] 月1回のコミュニティミーティング（またはランチ）に参加

**推奨条件**
- [ ] 既存のお客さんが数名いること（すぐ使い始められる）
- [ ] HACSのSNSへの出演・コンテンツ協力OK
- [ ] ドロップインイベントへの参加・協力

### HACSが守ること
- [ ] パートナーのスケジュールを優先確保する
- [ ] 他パートナーとのバッティングがないよう調整
- [ ] HACSのSNSでパートナーを紹介・発信
- [ ] 既存会員へのパートナー紹介

---

## 6. スペース利用のオペレーション

### 予約・管理
- 事前予約制（最低3日前まで）
- カレンダー共有ツールで管理（Googleカレンダー等）
- 緊急キャンセルは前日17時まで

### 利用時間（確定）
- 日曜・月曜：終日
- 火〜金：7:00〜9:30
- 土曜：17:00〜
- 上記時間外は原則不可（要事前相談）

### スペース利用後のルール
- 使用後の原状復帰（器具・マット等の片付け）
- 清掃の責任分担を明確化
