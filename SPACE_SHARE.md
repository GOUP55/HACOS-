# HACOS スペースシェア 条件設計

> フリーランサーが「ここをホームにしたい」と思える仕組みをつくる

---

## 1. HACSがスペースシェアで提供するもの（価値の整理）

パートナーが受け取るのは「場所」だけではない。

| 提供価値 | 内容 |
|---------|------|
| スペース・設備 | トレーニング機器・環境一式 |
| ブランド | HACOS の世界観・コンセプト |
| コミュニティ | 他のパートナーとの繋がり・相互紹介 |
| 集客サポート | HACSのSNS・既存会員への紹介 |
| イベント参加 | 日曜ドロップイン等のコラボ機会 |
| 信頼感 | 「ちゃんとした場所」でやっているという安心感 |

---

## 2. パートナー対象ジャンル

HACSのコンセプト「体を整え、体験する」に合うサービス。

**優先ジャンル**
- ヨガ・ピラティスインストラクター
- パーソナルトレーナー（専門特化型）
- 整体・ボディケア施術者
- コンディショニング・ストレッチ専門

**拡張ジャンル（将来）**
- サウナ・リトリート企画者
- 栄養士・食事コーチング
- メンタル・マインドフルネス系

---

## 3. スペースシェアの料金モデル（3案比較）

### プランA：売上シェア型（推奨：初期フェーズ）

```
HACSがもらう = パートナーの売上 × 20〜25%
```

| 項目 | 内容 |
|------|------|
| メリット | 参入障壁が低い・パートナーが成功するとHACSも潤う |
| デメリット | 収入が読みにくい・透明性の担保が必要 |
| 向いてる人 | まだ軌道に乗っていないフリーランサー・最初の仲間 |

**具体例：**
- 1セッション6,000円のヨガレッスン → HACOS受取 1,200〜1,500円
- 月10セッション → HACOS月収 12,000〜15,000円/人

---

### プランB：月額固定型

```
月額：30,000〜50,000円（利用頻度・曜日で変動）
```

| 曜日・頻度 | 月額（目安） |
|-----------|------------|
| 週1日（月4回）| 20,000〜25,000円 |
| 週2日（月8回）| 35,000〜40,000円 |
| 週3日以上 | 要相談 |

| 項目 | 内容 |
|------|------|
| メリット | 収入が安定・管理がシンプル |
| デメリット | パートナーにとって負担になりやすい（特に初期）|
| 向いてる人 | すでにある程度お客さんがいるトレーナー |

---

### プランC：ハイブリッド型（中期目標）

```
月額最低保証（10,000〜15,000円）+ 売上シェア15%
```

| 項目 | 内容 |
|------|------|
| メリット | 双方にとってフェア・長期関係に向いている |
| デメリット | 最初の設計が少し複雑 |
| 向いてる人 | 関係が深まってきたパートナー |

---

## 4. 推奨：フェーズ別の進め方

### フェーズ1（今〜3ヶ月）：最初の仲間を作る段階
**→ プランA（売上シェア20%）を基本にする**

理由：
- ハードルを下げて「まず来てもらう」ことが最優先
- お互いの信頼関係を築く期間
- HACOS側もどのくらい使われるか様子を見られる

### フェーズ2（3〜6ヶ月）：関係が安定してきたら
**→ プランAからプランCへ移行、または個別相談**

### フェーズ3（6ヶ月以降）：コミュニティが自走し始めたら
**→ プランBまたはCを標準化、新規はプランAから**

## 5. 基本条件・ルール

### パートナーに求めること

**必須条件**
- [ ] HACSのコンセプト・雰囲気に共感できること
- [ ] 自身のSNS・発信があること（相互発信が基本）
- [ ] 利用時のスペース・設備の管理責任を持てること
- [ ] 月1回のコミュニティミーティング（またはランチ）に参加

**推奨条件**
- [ ] 既存のお客さんが数名いること（すぐ使い始められる）
- [ ] HACSのSNSへの出演・コンテンツ協力OK
- [ ] ドロップインイベントへの参加・協力

### HACSが守ること
- [ ] パートナーのスケジュールを優先確保する
- [ ] 他パートナーとのバッティングがないよう調整
- [ ] HACSのSNSでパートナーを紹介・発信
- [ ] 既存会員へのパートナー紹介

---

## 6. スペース利用のオペレーション

### 予約・管理
- 事前予約制（最低3日前まで）
- カレンダー共有ツールで管理（Googleカレンダー等）
- 緊急キャンセルは前日17時まで

### 利用時間
- 基本：営業時間内（7:00〜21:00目安）
- 延長・早朝は別途相談

### スペース利用後のルール
- 使用後の原状復帰（器具・マット等の片付け）
- 清掃の責任分担を明確化
